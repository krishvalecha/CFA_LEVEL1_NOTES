The Simple Linear Regression Model

★ The Population Regression Model
Yᵢ = b₀ + b₁Xᵢ + εᵢ,    i = 1, 2, …, n
Yᵢ = dependent variable (explained variable) for observation i — what we want to explain. Plotted on vertical axis (Y-axis).
Xᵢ = independent variable (explanatory variable) for observation i — what we use to explain Y. Plotted on horizontal axis (X-axis).
b₀ = intercept (population parameter) — value of Y when X = 0.
b₁ = slope coefficient (population parameter) — change in Y for a one-unit change in X.
εᵢ = error term — difference between the observed Yᵢ and the true population relationship. E(ε) = 0 by assumption.
b₀, b₁ are unknown population parameters. We estimate them as b̂₀, b̂₁ from sample data (hats = estimated values).
Dependent Variable (Y)
The variable whose variation we are trying to explain. Also called the explained variable, regressand, or response variable. Examples: ROA, stock return, inflation rate, company revenue.
Independent Variable (X)
The variable we use to explain the variation of Y. Also called the explanatory variable, regressor, or predictor variable. Examples: CAPEX, S&P 500 return, money supply growth.
Error Term (εᵢ)
The difference between the observed Yᵢ and the true (unobservable) population relationship b₀ + b₁Xᵢ. Represents all factors affecting Y other than X. Population concept — not directly observable.
Residual (eᵢ)
The difference between observed Yᵢ and the fitted value Ŷᵢ from the estimated regression: eᵢ = Yᵢ − Ŷᵢ. Sample concept — calculated from the data. The vertical distances from data points to the regression line. Sum of residuals = 0.
Simple vs. Multiple Regression
  • Simple linear regression (SLR): one independent variable X. Y = b₀ + b₁X + ε.
  • Multiple regression: two or more independent variables X₁, X₂, ... Xₖ.
  • SLR is the special case of k = 1 independent variable.
  • Also called OLS regression (ordinary least squares).
Cross-Sectional vs. Time-Series Data
  • Cross-sectional: many observations of X and Y at the SAME time period (e.g., different companies in 2023). Subscript i = 1, 2, …, n.
  • Time-series: many observations of X and Y at DIFFERENT time periods for the SAME entity (e.g., monthly returns over 10 years). Subscript t = 1, 2, …, T.
02

OLS Estimation — Calculating the Coefficients

The least squares criterion (OLS = Ordinary Least Squares): choose b̂₀ and b̂₁ to minimize the sum of squared residuals (SSE) — the sum of the squared vertical distances from each data point to the fitted regression line.

Least Squares Criterion — What We Minimize
SSE = Σᵢ(Yᵢ − Ŷᵢ)² = Σᵢ[Yᵢ − (b̂₀ + b̂₁Xᵢ)]² = Σᵢeᵢ² → MINIMIZE
We minimize the sum of SQUARED residuals (not absolute values) for mathematical tractability. Squaring penalizes large deviations more, prevents positive and negative residuals from cancelling, and yields a unique closed-form solution.
OLS Slope Coefficient (b̂₁) — Equation 5
b̂₁ = Covariance(Y, X) / Variance(X)

b̂₁ = Σᵢ(Yᵢ − Ȳ)(Xᵢ − X̄) / Σᵢ(Xᵢ − X̄)²
Numerator = sum of cross-products (captures how X and Y move together relative to their means).
Denominator = variation of X around its mean (sum of squared deviations of X).
Sign rule: if covariance(Y,X) > 0 → b̂₁ > 0 (positive slope). If covariance < 0 → b̂₁ < 0. Same sign as the correlation r.
Difference from correlation: correlation denominator = s_X × s_Y; slope denominator = s²_X (variance of X only).
OLS Intercept (b̂₀) — Equation 6
b̂₀ = Ȳ − b̂₁X̄
Always calculate the slope FIRST, then use it to find the intercept.
Important mathematical result: the fitted regression line ALWAYS passes through the point (X̄, Ȳ) — the means of X and Y.
Ȳ = sample mean of the dependent variable. X̄ = sample mean of the independent variable.
Fitted (Predicted) Value and Residual
Ŷᵢ = b̂₀ + b̂₁Xᵢ            ← predicted value of Y for observation i
eᵢ = Yᵢ − Ŷᵢ               ← residual for observation i

Key properties: Σeᵢ = 0 (residuals sum to zero) and Ȳ = Ȳ̂ (mean of actual Y = mean of fitted Y)
📐 ROA vs. CAPEX — Module Example: Computing the Coefficients

n = 6 companies. Dependent variable Y = ROA. Independent variable X = CAPEX.

Cov(Y, X) = Σ(Yᵢ−Ȳ)(Xᵢ−X̄)/(n−1) = 153.30/5 = 30.6600
Var(X) = Σ(Xᵢ−X̄)²/(n−1) = 122.64/5 = 24.528 → s²_X = 24.528

b̂₁ = 30.6600 / 24.528 = 1.25 (slope)
b̂₀ = Ȳ − b̂₁ × X̄ = 12.5 − 1.25 × 6.1 = 4.875 (intercept)
Estimated model: ROA = 4.875 + 1.25 × CAPEX

Also: r = Cov(Y,X) / (s_Y × s_X) = 30.66 / (6.9210 × 4.9526) = 0.8945. Check: R² = r² = 0.8945² = 0.8001 ✓

💡 Slope vs. Correlation — The Key Difference

Both b̂₁ and r have the same sign (both determined by the covariance of Y and X). But:
Slope: b̂₁ = Cov(Y,X) / Var(X) → denominator is variance of X only.
Correlation: r = Cov(Y,X) / (s_Y × s_X) → denominator is product of BOTH standard deviations.
Important link: b̂₁ = r × (s_Y / s_X). Correlation is unit-free (−1 to +1); slope has units of Y per unit of X.

03

Interpreting the Regression Coefficients

Intercept (b̂₀)
  • The predicted value of Y when X = 0.
  • May or may not be economically meaningful — if X = 0 is unrealistic (e.g., zero money supply), the intercept has no practical interpretation.
  • In ROA example: ROA = 4.875% if CAPEX = 0 (company makes no capital expenditures).
  • With an indicator variable (X = 0 or 1): intercept = mean of Y for the X = 0 group.
Slope (b̂₁)
  • The change in Y for a one-unit change in X.
  • Positive slope → X and Y move in the same direction.
  • Negative slope → X and Y move in opposite directions.
  • In ROA example: a 1% increase in CAPEX → 1.25% increase in ROA.
  • With an indicator variable (X = 0 or 1): slope = difference in means between the X = 1 group and the X = 0 group.
  • Residual units = same units as Y (euros if Y in euros, % if Y in %).

Indicator (Dummy) Variables as the Independent Variable

An indicator variable (dummy variable) takes only the values 0 or 1. Example: EARN = 1 if month has an earnings announcement; EARN = 0 otherwise.

Model: RETᵢ = b₀ + b₁EARNᵢ + εᵢ

  • Intercept b̂₀: predicted Y when X = 0 → mean return for non-earnings-announcement months.
  • Slope b̂₁: when X changes from 0 to 1 → difference in mean returns between earnings-announcement months and non-earnings-announcement months.
  • Hypothesis test of the slope tests whether there is a statistically significant difference in means between the two groups.
⚠️ Exam Trap: Intercept May Be Meaningless

Always check whether X = 0 is a realistic scenario. If X = GDP growth rate, the intercept (revenue growth when GDP growth = 0%) is meaningful. If X = money supply level, the intercept (Y when money supply = 0) is not realistic and has no meaningful interpretation. The CFA exam often tests whether you recognize when the intercept is or isn't interpretable.

04

Four Assumptions of the SLR Model (CLRM)

🧠 Mnemonic: LHIN (or "Learn How It's Normal")

Linearity · Homoskedasticity · Independence · Normality. These four assumptions must hold for OLS estimates to be valid, unbiased, and efficient (BLUE — Best Linear Unbiased Estimator).

# Assumption What It Means How to Detect Violation Consequence of Violation
1 Linearity The true relationship between Y and X is linear. X must be non-stochastic (non-random). Residual plot shows a pattern (e.g., curved shape, U-shape) vs. X. Residuals should look random, with no discernible pattern. Model is biased — it over/under-estimates Y at different ranges of X. Invalid results.
2 Homoskedasticity Variance of residuals is constant across all observations: E(εᵢ²) = σ²ε for all i. Residual plot shows residuals clustering into groups with different spreads, or a funnel/fan shape. Indicates multiple regimes with different variances. Heteroskedasticity: OLS standard errors are incorrect → t-tests and F-tests are invalid. Coefficients still unbiased but tests are unreliable.
3 Independence Observations (Y, X pairs) are independent of each other. Residuals are uncorrelated across observations. Residual plot shows a pattern over time (e.g., seasonal pattern, Q4 spike). Autocorrelation in residuals. Durbin-Watson test. Autocorrelation: parameter variances estimated incorrectly → invalid hypothesis tests. Affects time-series regressions most.
4 Normality Regression residuals are normally distributed. (NOT that Y or X must be normal — only the residuals.) QQ-plot of residuals departs from a straight line; histogram of residuals is skewed or fat-tailed. Outliers in data can severely distort the fitted line. t-tests and F-tests may be unreliable for small samples. For large n, CLT allows relaxation of this assumption.
💡 Reading Residual Plots — What Good Looks Like

Plot residuals on the Y-axis vs. the independent variable X on the X-axis. Good (no violations): residuals appear random, scattered evenly around zero, no discernible pattern, approximately equal spread across all values of X. Bad: any systematic pattern — curved shape (linearity violation), funnel shape widening with X (heteroskedasticity), seasonal pattern (autocorrelation).

⚠️ Heteroskedasticity Regime Example

If a regression spans two distinct economic regimes (e.g., normal interest rates vs. QE/artificially low rates), the residuals will have different variances in each regime. This is clearly visible in a residual plot — wider band for one period vs. another. The homoskedasticity assumption is violated. The solution is to model each regime separately.

💡 Normality & Large Samples — CLT to the Rescue

The normality assumption is most critical for small samples. For large samples, the Central Limit Theorem guarantees that the sampling distributions of b̂₀ and b̂₁ are approximately normal regardless of the residual distribution. So in large samples, we can often relax the normality requirement and still have valid t-tests and F-tests.

Decomposition of Variation: SST = SSR + SSE

★ The Fundamental Variance Decomposition
SST = SSR + SSE

SST = Σᵢ(Yᵢ − Ȳ)²      ← Total variation in Y (total sum of squares)
SSR = Σᵢ(Ŷᵢ − Ȳ)²      ← Explained variation (regression sum of squares)
SSE = Σᵢ(Yᵢ − Ŷᵢ)²      ← Unexplained variation (error/residual sum of squares)
SST = total variation of Y around its mean. Measures what we're trying to explain.
SSR = variation of the fitted values (Ŷ) around the mean of Y. Measures how much X explains.
SSE = variation of actual Y values around the fitted line. Measures what X did NOT explain.
Higher SSR / lower SSE → better model fit. Goal: make SSR as large as possible (explain more of SST).
📐 ROA Example: Decomposition of Variation
SST = 239.50 ← total variation in ROA we want to explain
SSR = 191.63 ← variation explained by CAPEX
SSE = 47.88 ← variation NOT explained by CAPEX
Check: 191.63 + 47.88 = 239.51 ≈ 239.50 ✓ (rounding)
R² = SSR/SST = 191.63/239.50 = 0.8001 → 80.01% of ROA variation explained by CAPEX
🧠 Memory Trick: SST = SSR + SSE

T = R + E: Total = Regression (explained) + Error (unexplained). Think: "Total is Really Explained + Errors." The numerator of R² is SSR (what the Regression explained). The denominator is SST (the Total variation). SSE is what you couldn't explain — the Errors left over.

ANOVA Table for Simple Linear Regression

Source Sum of Squares Degrees of Freedom Mean Square F-Statistic
Regression SSR = Σ(Ŷᵢ−Ȳ)² k = 1 MSR = SSR/1 = SSR F = MSR/MSE
Error SSE = Σ(Yᵢ−Ŷᵢ)² n − k − 1 = n − 2 MSE = SSE/(n−2)
Total SST = Σ(Yᵢ−Ȳ)² n − 1
Standard Error of the Estimate (sₑ) — Square Root of MSE
sₑ = √MSE = √[ SSE / (n−2) ] = √[ Σ(Yᵢ−Ŷᵢ)² / (n−2) ]
Also called: standard error of the regression, root mean square error (RMSE).
Interpretation: absolute measure of the average distance between actual Y values and the fitted regression line. Units = same as Y.
Lower sₑ → better model fit. Unlike R², sₑ is an absolute (not relative) measure of fit.
Uses n−2 degrees of freedom because two parameters (b₀, b₁) are estimated.
Used to: (1) calculate prediction intervals, (2) calculate standard error of slope and intercept, (3) assess goodness of fit.
ANOVA Table for ROA Example (Module Numbers)

Regression 191.63   1   191.63   F = 191.63/11.97 = 16.01
Error        47.88    4    11.97
Total       239.50   5

sₑ = √11.97 = 3.46 (standard error of the estimate)
Critical F(5%, 1, 4) = 7.71. Since 16.01 > 7.71 → Reject H₀: b₁ = 0 ✓
Note: n=6, k=1, so df_error = 6−1−1 = 4, df_total = 6−1 = 5. Variance of Y = SST/(n−1) = 239.50/5 = 47.90.
💡 Key Relationship: t² = F in Simple Linear Regression

In SLR, the F-statistic for testing whether b₁ = 0 equals the square of the t-statistic for the same test: t² = F. In the ROA example: t = 4.001, F = 16.01, and indeed 4.001² ≈ 16.01. This means the F-test and the t-test on the slope give identical conclusions for SLR. In multiple regression, they can differ.

07

Goodness of Fit Measures

R² (Coefficient of Determination)

R² = SSR/SST = 1 − SSE/SST

  • Range: 0% to 100%.
  • % of variation in Y explained by X.
  • Descriptive — NOT a statistical test.
  • In SLR: R² = r² (square of Pearson correlation).
  • R² = 0.80 → X explains 80% of variation in Y.
  • Higher R² = better fit. But not sufficient alone.
F-Statistic (Test of Overall Fit)

F = MSR/MSE = (SSR/1)/(SSE/(n−2))

  • Tests H₀: b₁ = 0 (slope = 0 → no linear relationship).
  • F-distribution with df = (1, n−2) for SLR.
  • Always right-tail test (rejection region on right).
  • Reject H₀ if F > F_critical.
  • Statistical test (unlike R²).
  • In SLR: F = t² where t is the slope t-statistic.
sₑ (Standard Error of Estimate)

sₑ = √(SSE/(n−2)) = √MSE

  • Absolute measure of fit (has units of Y).
  • Average distance of actual Y from the fitted line.
  • Lower sₑ = better fit.
  • Unlike R² and F, not relative — allows direct comparison of models with the same Y variable.
  • Used in prediction intervals and coefficient tests.
⚠️ Exam Trap: R² is Not a Statistical Test

R² tells you how much variation is explained, but it does NOT tell you whether the relationship is statistically significant. A model can have high R² but statistically insignificant slope (e.g., with few observations). Always use the F-statistic or t-test on the slope to determine statistical significance — R² alone is insufficient.

Hypothesis Tests on Regression Coefficients

A. Testing the Slope Coefficient (b₁)

t-Test for the Slope Coefficient
t = (b̂₁ − B₁) / s_b̂₁        df = n − k − 1 = n − 2

Standard error of the slope: s_b̂₁ = sₑ / √[Σ(Xᵢ−X̄)²]
b̂₁ = estimated slope from OLS. B₁ = hypothesized population slope value (often B₁ = 0 to test if relationship exists).
s_b̂₁ = standard error of the slope coefficient. Depends on sₑ (model fit) and Σ(Xᵢ−X̄)² (variation of X).
Key insight: greater variability of X → lower s_b̂₁ → larger t-statistic → easier to reject H₀. More spread in X gives better ability to estimate the slope.
Most common test: H₀: b₁ = 0 vs. Hₐ: b₁ ≠ 0 (two-sided). Can also test one-sided or against non-zero hypothesized value.
Note: t-test for slope (H₀: b₁ = 0) and t-test for correlation (H₀: ρ = 0) give the SAME calculated t-statistic in SLR.
📐 ROA Example: Testing H₀: b₁ = 0 (two-tailed, α = 5%)

b̂₁ = 1.25, B₁ = 0 (test if slope differs from zero). df = 6 − 2 = 4. Critical values: ±2.776.

s_b̂₁ = sₑ / √Σ(Xᵢ−X̄)² = 3.46 / √122.64 = 3.46 / 11.074 = 0.312398
t = (1.25 − 0) / 0.312398 = 4.001
|4.001| > 2.776 → REJECT H₀: b₁ = 0. CAPEX is a significant explanatory variable of ROA.

p-value = 0.016. Since 0.016 < 0.05, reject H₀ at 5% significance. R² = 0.8001. F = 16.01 = 4.001² ✓

B. Testing the Intercept (b₀)

t-Test for the Intercept
t = (b̂₀ − B₀) / s_b̂₀        df = n − 2

s_b̂₀ = sₑ × √[ 1/n + X̄² / Σ(Xᵢ−X̄)² ]
B₀ = hypothesized population intercept value. Standard error of intercept involves sₑ, the mean of X (X̄), and the variation of X.
Less commonly tested than the slope. The intercept test is most meaningful when X = 0 is a realistic scenario.

C. Key Relationships Between Tests in SLR

  • t-test for slope b₁ = 0 and t-test for correlation ρ = 0 produce the same calculated t-statistic in SLR. Both use t = r√(n−2)/√(1−r²) with df = n−2.
  • t² = F: The square of the t-statistic for testing b₁ = 0 equals the F-statistic for the overall regression test in SLR.
  • Both t-test (slope) and F-test (fit) test the same hypothesis in SLR: H₀: b₁ = 0 — they will always agree on accept/reject.
  • p-value interpretation: if p-value = 0.016, there is a 1.6% probability of observing a test statistic this extreme if H₀ is true. Reject H₀ if p-value < α.

Prediction and Prediction Intervals

Point Forecast of Dependent Variable (Equation 23)
Ŷf = b̂₀ + b̂₁ × Xf
Xf = the forecasted (given) value of the independent variable.
Ŷf = the predicted value of Y for that Xf value.
Example (ROA): if CAPEX = 6%, then Ŷf = 4.875 + 1.25 × 6 = 12.375%.
Standard Error of the Forecast (Equation 24)
sf = sₑ × √[ 1 + 1/n + (Xf − X̄)² / Σᵢ(Xᵢ−X̄)² ]
The "1 +" inside the square root makes sf > sₑ always — a prediction interval is always wider than a confidence interval for the mean of Y at Xf.
Three factors that affect sf:
(1) Better model fit (smaller sₑ) → smaller sf → narrower interval.
(2) Larger sample size n → smaller sf → narrower interval.
(3) Closer Xf to X̄ → smaller sf → narrower interval. Predictions far from X̄ (extrapolation) are less precise.
Prediction Interval for Ŷf
Prediction interval = Ŷf ± t_critical(α/2, n−2) × sf
t_critical uses df = n − 2 (same as for testing coefficients in SLR).
The prediction interval reflects uncertainty from TWO sources: (1) the regression line itself is an estimate (not perfect), and (2) individual Y values scatter around the line (residual variance).
A confidence interval for the mean of Y at Xf would use sf without the "1 +" and would be narrower than the prediction interval.
1
Calculate the point forecast: Ŷf = b̂₀ + b̂₁ × Xf using the estimated regression equation.
2
Choose a significance level α for the prediction interval (e.g., α = 5% for 95% PI).
3
Find t_critical with df = n−2 at α/2 (two-tailed) from the t-table.
4
Compute the standard error of the forecast: sf = sₑ × √[1 + 1/n + (Xf−X̄)²/Σ(Xᵢ−X̄)²].
5
Construct the (1−α)% prediction interval: [Ŷf − t × sf, Ŷf + t × sf].
⚠️ Exam Trap: Prediction Interval vs. Confidence Interval

A prediction interval gives the range for a single new observation of Y (wider). A confidence interval for the conditional mean of Y gives the range for the mean of Y at a given X (narrower — no "1 +" term in the variance formula). The exam may ask which is wider or for which situation each is appropriate. Prediction interval is always wider than the corresponding confidence interval for the mean.

Functional Forms for Non-Linear Data

Not all economic/financial relationships are linear. When the scatter plot or residuals show a non-linear pattern, we can transform the variables using logarithms to apply linear regression. Three common forms — all use the natural log (ln):

Model Name Equation Y Form X Form Slope Interpretation Finance Use
Lin-Lin (Standard SLR) Y = b₀ + b₁X Linear Linear Absolute change in Y per 1-unit absolute change in X. Most general cases where relationship is linear.
Log-Lin ln Y = b₀ + b₁X Log Linear Relative (%) change in Y per 1-unit absolute change in X. (b̂₁ ≈ % change in Y) Exponential growth (revenues, populations); Y grows at a constant rate per unit X.
Lin-Log Y = b₀ + b₁ ln X Linear Log Absolute change in Y per 1% relative change in X. (b̂₁/100 = unit change per 1% change in X) Diminishing returns — operating profit margin vs. log(unit sales); Y grows slowly as X increases.
Log-Log (Double-Log) ln Y = b₀ + b₁ ln X Log Log Elasticity: % change in Y per 1% change in X. (b̂₁ = elasticity of Y with respect to X) Elasticity analysis — revenue vs. advertising; b̂₁ = revenue elasticity w.r.t. ad spend.

How to Choose the Right Functional Form

1
Examine the scatter plot of Y vs. X. If it appears non-linear (curved), a log transformation may be appropriate.
2
Try different functional forms and compare goodness-of-fit measures: R², F-statistic, and sₑ. Higher R², higher F, lower sₑ → better fit.
3
Examine residual plots. The correct model should show random residuals with no pattern. A pattern means the functional form is still misspecified.
4
Forecasting with log-lin model: if ln Ŷf = b̂₀ + b̂₁Xf = −2, then Ŷf = e^(−2) = 0.1353. Must back-transform using the antilog (exponential function).
⚠️ Exam Trap: Cannot Compare R² Across Different Y Forms

You cannot directly compare R² or F-statistic between a lin-lin model (Y = b₀ + b₁X) and a log-lin model (ln Y = b₀ + b₁X) because the dependent variable is in different forms. To compare these, look at the standard error of the estimate sₑ and residual plots. You CAN compare sₑ between lin-lin and lin-log models (same Y variable). You CANNOT compare sₑ between lin-lin and log-lin models.

🧠 Memory Trick: Which Variable is Logged?

Log-Lin: the LOG is the dependent (Y) variable, the LINear is X. So b̂₁ = relative change in Y / absolute change in X. Lin-Log: LINear is Y, LOG is X. So b̂₁ = absolute change in Y / relative change in X. Log-Log: both logged → b̂₁ = elasticity (relative/relative). The name tells you the order: (Y form)-(X form). Log-Lin = log Y, linear X.

★

Exam Checklist — Must-Know Points

  • ✅ SLR model: Yᵢ = b₀ + b₁Xᵢ + εᵢ. Y = dependent (explained); X = independent (explanatory); ε = error term (population concept). b₀, b₁ are unknown population parameters estimated by b̂₀, b̂₁.
  • ✅ OLS criterion: minimizes SSE = Σ(Yᵢ−Ŷᵢ)² = Σeᵢ². Fitted line minimizes sum of SQUARED vertical distances to data points.
  • ✅ Slope formula: b̂₁ = Cov(Y,X)/Var(X) = Σ(Yᵢ−Ȳ)(Xᵢ−X̄) / Σ(Xᵢ−X̄)². Same sign as correlation r. Calculate slope FIRST, then intercept.
  • ✅ Intercept formula: b̂₀ = Ȳ − b̂₁X̄. The regression line always passes through (X̄, Ȳ). Σeᵢ = 0.
  • ✅ Intercept interpretation: predicted Y when X = 0. May be meaningless if X = 0 is unrealistic.
  • ✅ Slope interpretation: change in Y for one-unit change in X. With dummy variable X: slope = difference in group means (X=1 vs X=0).
  • ✅ Four CLRM assumptions (LHIN): Linearity, Homoskedasticity, Independence, Normality. All concern the RESIDUALS (except linearity which also concerns X being non-random).
  • ✅ Detecting violations via residual plots: curved pattern = linearity violated; funnel/fan shape = heteroskedasticity; seasonal pattern = autocorrelation; outliers = normality concern.
  • ✅ Heteroskedasticity: different variance of residuals across observations. Detected by residuals clustering in groups. Makes standard errors invalid even though coefficients are unbiased.
  • ✅ SST = SSR + SSE. SST = total variation in Y. SSR = explained (by regression). SSE = unexplained (residual). Goal: maximize SSR, minimize SSE.
  • ✅ R² = SSR/SST. Range 0–100%. In SLR: R² = r² (square of Pearson correlation). Descriptive, NOT a statistical test. Does not confirm statistical significance alone.
  • ✅ F-statistic = MSR/MSE = SSR/(SSE/(n−2)). Tests H₀: b₁ = 0. F-distribution with df = (1, n−2). Right-tail test only. In SLR: F = t² (where t tests H₀: b₁ = 0).
  • ✅ Standard error of estimate (sₑ): sₑ = √MSE = √(SSE/(n−2)). Absolute measure of fit in units of Y. Lower sₑ = better fit. Used in prediction intervals and coefficient tests.
  • ✅ t-test for slope: t = (b̂₁ − B₁)/s_b̂₁, df = n−2. s_b̂₁ = sₑ/√Σ(Xᵢ−X̄)². More variable X → lower s_b̂₁ → larger t-stat. Test for H₀: b₁ = 0 and test for H₀: ρ = 0 give the same t-statistic.
  • ✅ Point forecast: Ŷf = b̂₀ + b̂₁Xf. Prediction interval: Ŷf ± t(α/2, n−2) × sf. Prediction interval is wider when: Xf far from X̄, small n, large sₑ.
  • ✅ sf > sₑ always (due to the "1 +" term). Prediction intervals are wider than confidence intervals for the conditional mean of Y.
  • ✅ Log-Lin: ln Y = b₀ + b₁X → b̂₁ = relative change in Y per absolute change in X. Lin-Log: Y = b₀ + b₁ ln X → b̂₁ = absolute change in Y per relative change in X. Log-Log: ln Y = b₀ + b₁ ln X → b̂₁ = elasticity (% change in Y per % change in X).
  • ✅ Cannot compare R² across models with different Y forms (e.g., lin-lin vs. log-lin). Use sₑ and residual plots instead when Y forms differ.
  • ✅ Key to selecting functional form: examine R², F-statistic, sₑ, and residual plots. Correct form → random residuals, no pattern.
★ COMPLETE FORMULA QUICK REFERENCE ★
═══ THE MODEL ═══
Population: Yᵢ = b₀ + b₁Xᵢ + εᵢ
Estimated: Ŷᵢ = b̂₀ + b̂₁Xᵢ      eᵢ = Yᵢ − Ŷᵢ (residual)   Σeᵢ = 0

═══ OLS COEFFICIENTS ═══
b̂₁ = Σ(Yᵢ−Ȳ)(Xᵢ−X̄) / Σ(Xᵢ−X̄)²     (slope — compute FIRST)
b̂₀ = Ȳ − b̂₁X̄                           (intercept — compute SECOND)

═══ SUMS OF SQUARES ═══
SST = Σ(Yᵢ−Ȳ)²   df = n−1     (total variation)
SSR = Σ(Ŷᵢ−Ȳ)²   df = k = 1   (explained by regression)
SSE = Σ(Yᵢ−Ŷᵢ)²  df = n−2     (unexplained / residual)
SST = SSR + SSE

═══ GOODNESS OF FIT ═══
R² = SSR/SST = 1 − SSE/SST    (In SLR: R² = r²)
MSR = SSR/1 = SSR              MSE = SSE/(n−2)
F = MSR/MSE   df=(1,n−2)     (H₀:b₁=0; right-tail; In SLR: F = t²)
sₑ = √MSE = √(SSE/(n−2))    (standard error of estimate)

═══ TESTING SLOPE ═══
s_b̂₁ = sₑ / √Σ(Xᵢ−X̄)²
t = (b̂₁ − B₁) / s_b̂₁      df = n−2    (B₁ = 0 most common)
Note: Same t-stat as testing H₀: ρ = 0. In SLR: t² = F.

═══ TESTING INTERCEPT ═══
s_b̂₀ = sₑ × √[1/n + X̄²/Σ(Xᵢ−X̄)²]
t = (b̂₀ − B₀) / s_b̂₀     df = n−2

═══ PREDICTION ═══
Ŷf = b̂₀ + b̂₁Xf
sf = sₑ × √[1 + 1/n + (Xf−X̄)²/Σ(Xᵢ−X̄)²]
PI = Ŷf ± t(α/2, n−2) × sf

═══ FUNCTIONAL FORMS ═══
Lin-Lin: Y = b₀+b₁X      → b₁ = ΔY per ΔX
Log-Lin: lnY = b₀+b₁X   → b₁ = %ΔY per ΔX (relative Y, absolute X)
Lin-Log: Y = b₀+b₁lnX   → b₁ = ΔY per %ΔX (absolute Y, relative X)
Log-Log: lnY = b₀+b₁lnX → b₁ = elasticity (%ΔY / %ΔX)
LHIN assumptions: Linearity (no pattern in residuals), Homoskedasticity (constant residual variance), Independence (no autocorrelation), Normality (residuals normal — can relax for large n via CLT).
F = t² in SLR; testing slope = testing correlation = same t-statistic. R² is descriptive, not a test.
Prediction interval > Confidence interval for mean (extra "1 +" term in sf). Wider when Xf far from X̄, small n, or large sₑ.
Log-Lin: forecast must be back-transformed: if ln Ŷf = c, then Ŷf = eᶜ. Cannot compare R² across models with different Y forms.
