01

Key Definitions — At a Glance

Portfolio Expected Return — E(Rp)
A weighted average of the expected returns on the individual securities, using each security's portfolio weight as the weight. The weights must sum to 1.
Portfolio Variance — σ²(Rp)
Measures expected investment risk. Depends on both the individual asset variances AND the covariances between assets. NOT simply a weighted average of individual variances.
Covariance — Cov(Rᵢ, Rⱼ)
The probability-weighted average of the cross-products of each asset's deviation from its own expected value. Measures the direction and degree of the co-movement of two returns. Can be negative, zero, or positive. No fixed range.
Correlation — ρ(Rᵢ, Rⱼ)
Covariance standardized by dividing by the product of the two standard deviations. Dimensionless (no units). Bounded between −1 and +1. Measures the strength and direction of linear association.
Shortfall Risk
The risk that portfolio value (or return) will fall below some minimum acceptable level (R_L) over some time horizon. Focuses only on downside risk, unlike standard deviation which captures both upside and downside.
Safety-First Ratio (SFRatio)
Measures how far the expected portfolio return is above the threshold level R_L, in standard deviation units. The higher the SFRatio, the lower the probability of falling below R_L.
Joint Probability Function — P(X, Y)
Gives the probability of joint occurrences of values of two random variables X and Y. For example, P(X=3, Y=2) = probability that X equals 3 AND Y equals 2 simultaneously.
02

Portfolio Expected Return

★ Portfolio Expected Return Formula
E(Rp) = w₁E(R₁) + w₂E(R₂) + ... + wₙE(Rₙ)

E(Rp) = Σ wᵢ · E(Rᵢ)  (summed from i = 1 to n)
wᵢ = weight of asset i in the portfolio (proportion of total portfolio value in currency units).
Constraint: Σ wᵢ = 1 (weights must sum to 1; can include short positions with negative weights).
E(Rᵢ) = expected return on asset i.
Portfolio expected return IS a simple weighted average of individual expected returns.
Portfolio variance is NOT — it also requires covariance terms.
📊 Worked Example — 2-Asset Portfolio Expected Return

Portfolio: 60% in Stock A (E(Rₐ) = 12%), 40% in Stock B (E(R_b) = 8%)

E(Rp) = 0.60 × 12% + 0.40 × 8%
= 7.2% + 3.2%
= 10.4%
→ Portfolio Expected Return = 10.4%
💡 Key Insight — E(Rp) is Easy; Variance is the Challenge

Once you have individual expected returns, portfolio E(Rp) is trivial — just a weighted sum. The real complexity of portfolio mathematics lies in the variance calculation, which requires covariances between every pair of assets. This is why covariance is so central to portfolio theory.

03

Covariance

★ Covariance — Population (Forward-Looking)
Cov(Rᵢ, Rⱼ) = E[(Rᵢ − ERᵢ)(Rⱼ − ERⱼ)]

Alternative notations: σ(Rᵢ, Rⱼ) and σᵢⱼ
= probability-weighted average of the cross-products of each return's deviation from its own expected value.
"Population" covariance = forward-looking (uses expected values and probabilities, not historical data).
★ Covariance — Sample (Historical)
Cov(Rᵢ, Rⱼ) = Σ (Rᵢ,ₜ − R̄ᵢ)(Rⱼ,ₜ − R̄ⱼ) / (n − 1)
Based on past data of n observations. Uses sample means R̄ᵢ and R̄ⱼ instead of expected values.
Denominator is (n − 1) — same as sample variance formula (degrees of freedom adjustment).

Interpreting the Sign of Covariance

Cov < 0 (Negative)

When return on one asset is above its expected value, the return on the other tends to be below its expected value. Average inverse relationship. Excellent for diversification — reduces portfolio variance.

Cov = 0 (Zero)

Returns on the two assets are unrelated — no linear association. Neither helps nor hurts diversification beyond simply holding more assets. (NOTE: zero covariance does NOT necessarily mean independence.)

Cov > 0 (Positive)

Returns tend to move together — both above or both below expected values at the same time. Average direct relationship. Adds to portfolio variance — reduces diversification benefit.

💡 Own Covariance = Variance

The covariance of a random variable with itself is its own variance:
Cov(R, R) = E{[R − E(R)][R − E(R)]} = E{[R − E(R)]²} = σ²(R)

This is why the diagonal entries of a covariance matrix are variances. The covariance formula generalizes the variance formula.

⚠️ Covariance Has No Fixed Range — That's Why We Need Correlation

Covariance can range from −∞ to +∞. Its magnitude depends on the scale of the returns and is not directly interpretable as "strong" or "weak." Correlation solves this by standardizing covariance to a −1 to +1 range.

04

Correlation

★ Correlation Formula & Rearrangement
ρ(Rᵢ, Rⱼ) = Cov(Rᵢ, Rⱼ) / [σ(Rᵢ) × σ(Rⱼ)]

Alternative notations: Corr(Rᵢ, Rⱼ) and ρᵢⱼ

Rearranged (very useful — gives covariance from correlation):
Cov(Rᵢ, Rⱼ) = ρ(Rᵢ, Rⱼ) × σ(Rᵢ) × σ(Rⱼ)
Correlation is a PURE NUMBER — no units (the units cancel in the division).
Range: −1 ≤ ρ ≤ +1. Always bounded, unlike covariance.
The rearrangement Cov = ρ × σᵢ × σⱼ is extremely useful on the exam — plugs directly into the 2-asset variance formula.
Correlation ValueInterpretationPortfolio Impact
ρ = +1Perfect positive linear relationship; assets move exactly togetherNO diversification benefit; portfolio variance = weighted average of individual variances
0 < ρ < +1Positive linear relationshipSome diversification benefit; portfolio variance < weighted average of variances
ρ = 0No linear relationship (uncorrelated)More diversification benefit; full unsystematic risk reduction possible
−1 < ρ < 0Negative linear relationshipStrong diversification benefit
ρ = −1Perfect negative linear relationship; assets move exactly oppositeMAXIMUM diversification benefit; can potentially eliminate all risk
💡 Correlation vs. Covariance — The Key Difference

Both measure linear association between two variables. Covariance: has units (e.g., %²), unbounded, tells direction but not strength. Correlation: no units, bounded [−1, +1], tells both direction AND standardized strength. Use covariance in portfolio variance calculations; use correlation for interpretation and comparison across pairs of assets.

05

Portfolio Variance — The Core Formula

★ 2-Asset Portfolio Variance (Most Tested)
σ²(Rp) = w₁²σ²(R₁) + w₂²σ²(R₂) + 2w₁w₂Cov(R₁, R₂)

Substituting Cov = ρ × σ₁ × σ₂:
σ²(Rp) = w₁²σ²₁ + w₂²σ²₂ + 2w₁w₂ρ₁₂σ₁σ₂

Standard deviation: σ(Rp) = √σ²(Rp)
3 terms: 2 variance terms (one for each asset) + 1 covariance term (the interaction).
The covariance term is multiplied by 2 because Cov(R₁,R₂) = Cov(R₂,R₁) — it appears twice in the full expansion.
The covariance term is how correlation affects portfolio risk. Lower ρ → lower portfolio variance.
★ 3-Asset Portfolio Variance (Know the Structure)
σ²(Rp) = w₁²σ²(R₁) + w₂²σ²(R₂) + w₃²σ²(R₃)
           + 2w₁w₂Cov(R₁,R₂)
           + 2w₁w₃Cov(R₁,R₃)
           + 2w₂w₃Cov(R₂,R₃)

= 3 variance terms + 6 covariance terms (3 unique pairs × 2)
For n assets: n variance terms + n(n−1) covariance terms [i.e., n² − n off-diagonal entries].
Example: 20 assets → 20 variance terms + 380 covariance terms.
As n grows, covariance terms dominate — covariance becomes MORE important than individual variances.
★ General n-Asset Portfolio Variance
σ²(Rp) = Σᵢ Σⱼ wᵢwⱼCov(Rᵢ, Rⱼ)  (i and j each from 1 to n)

Note: When i = j, Cov(Rᵢ, Rᵢ) = σ²(Rᵢ) (own variance on diagonal)
This double-sum formula is the most general and compact form.
Diagonal terms (i = j): variance of each asset.
Off-diagonal terms (i ≠ j): covariance between each pair of assets.
📊 Worked Example — 2-Asset Portfolio Variance

w₁ = 60%, w₂ = 40%, σ₁ = 15%, σ₂ = 10%, ρ₁₂ = 0.30

Step 1: Cov(R₁,R₂) = ρ₁₂ × σ₁ × σ₂ = 0.30 × 15% × 10% = 4.5 (%²)

Step 2: σ²(Rp) = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(R₁,R₂)
= (0.60)²(225) + (0.40)²(100) + 2(0.60)(0.40)(4.5)
= (0.36)(225) + (0.16)(100) + (0.48)(4.5)
= 81 + 16 + 2.16
= 99.16 (%²)

Step 3: σ(Rp) = √99.16 = 9.96%
→ Portfolio Variance = 99.16 (%²) | Portfolio Std Dev ≈ 9.96%

Note: Simple weighted-average std dev would be 0.60×15% + 0.40×10% = 13%, far higher. Diversification (ρ < 1) reduces portfolio risk from 13% to 9.96% — the power of diversification.

💡 Diversification Benefit — The Key Insight

When ρ = +1: σ(Rp) = w₁σ₁ + w₂σ₂ (weighted average — no benefit from combining)
When ρ < +1: σ(Rp) < w₁σ₁ + w₂σ₂ (portfolio risk is LESS than weighted average — diversification works)
When ρ = −1: Can construct zero-variance portfolio (weights: w₁ = σ₂/(σ₁+σ₂), w₂ = σ₁/(σ₁+σ₂))

The lower the correlation, the greater the diversification benefit — this is the fundamental principle of modern portfolio theory.

06

Covariance Matrix & Correlation Matrix

A covariance matrix (also called a variance-covariance matrix) organizes all the variance and covariance terms needed to compute portfolio variance. For an n-asset portfolio, the matrix has n rows and n columns:

  • Diagonal entries (i = j): Variances σ²(Rᵢ) = Cov(Rᵢ, Rᵢ)
  • Off-diagonal entries (i ≠ j): Covariances Cov(Rᵢ, Rⱼ)
  • Matrix is symmetric: Cov(Rᵢ, Rⱼ) = Cov(Rⱼ, Rᵢ), so the upper-right and lower-left triangles are mirrors
  • A complete covariance matrix contains ALL statistical data needed to compute portfolio variance

Example — 2-Asset Covariance Matrix

Asset 1Asset 2
Asset 1 σ²(R₁) = Cov(R₁,R₁) Cov(R₁,R₂)
Asset 2 Cov(R₂,R₁) σ²(R₂) = Cov(R₂,R₂)

Converting Between Covariance and Correlation Matrices

Covariance → Correlation

ρᵢⱼ = Cov(Rᵢ,Rⱼ) / [σ(Rᵢ) × σ(Rⱼ)]

Diagonal of correlation matrix = 1.0 always (each variable is perfectly correlated with itself).

Need: σᵢ = √Cov(Rᵢ,Rᵢ) from the diagonal of the covariance matrix.

Correlation → Covariance

Cov(Rᵢ,Rⱼ) = ρᵢⱼ × σ(Rᵢ) × σ(Rⱼ)

Used to reconstruct the covariance matrix from a correlation matrix + standard deviations.

On the exam: often given correlation matrix + std devs → must compute covariances for variance formula.

💡 Number of Terms in an n-Asset Portfolio Variance

Total terms in covariance matrix = n² (n rows × n columns)
Diagonal (variance) terms = n
Off-diagonal (covariance) terms = n² − n = n(n−1)
Unique covariance pairs = n(n−1)/2 (half the off-diagonal, by symmetry)

Example: 20 assets → 20 variances + 380 covariances (20 × 19) → covariance terms outnumber variance terms 19:1.
This is why as portfolio size grows, individual asset risk matters less; covariances (correlations) dominate.

07

Computing Covariance from a Joint Probability Function

When historical data is unavailable, covariance can be computed using a joint probability function P(Rₐ, R_b) that gives the probability of each combination of outcomes for two assets A and B.

★ Covariance from Joint Probability Function
Cov(Rₐ, R_b) = Σᵢ Σⱼ P(Rₐ,ᵢ, R_b,ⱼ) × (Rₐ,ᵢ − ERₐ) × (R_b,ⱼ − ER_b)
Steps:
1. Calculate E(Rₐ) and E(R_b) from the joint probability function (sum all outcomes weighted by their marginal probabilities).
2. For each joint outcome (Rₐ,ᵢ, R_b,ⱼ), compute the deviation cross-product: (Rₐ,ᵢ − ERₐ)(R_b,ⱼ − ER_b).
3. Multiply each cross-product by its joint probability P(Rₐ,ᵢ, R_b,ⱼ).
4. Sum all the probability-weighted cross-products → Cov(Rₐ, R_b).
5. Once you have Cov and both σ values, compute ρ = Cov / (σₐ × σ_b).
📊 Worked Example — Covariance from Joint Probability

Simple 2-scenario joint probability table for stocks A and B:

Scenario P Rₐ R_b
─────────────────────────────
Good 0.60 15% 10%
Bad 0.40 -5% -2%

Step 1 — E(Rₐ) = 0.60(15%) + 0.40(-5%) = 9.0% − 2.0% = 7.0%
Step 1 — E(R_b) = 0.60(10%) + 0.40(-2%) = 6.0% − 0.8% = 5.2%

Step 2/3/4 — Cov(Rₐ, R_b):
= 0.60 × (15%−7%) × (10%−5.2%) + 0.40 × (-5%−7%) × (-2%−5.2%)
= 0.60 × (8%) × (4.8%) + 0.40 × (-12%) × (-7.2%)
= 0.60 × 38.4 + 0.40 × 86.4
= 23.04 + 34.56
= 57.6 (%²)
→ Cov(Rₐ, R_b) = 57.6 (%²)  [positive → returns move together]
08

Independence vs. Uncorrelatedness

Two random variables X and Y are independent if and only if: P(X, Y) = P(X) × P(Y)

Given independence, joint probabilities are simply the product of marginal probabilities: P(X=3, Y=2) = P(X=3) × P(Y=2).

PropertyIndependentUncorrelated (ρ = 0)
Definition P(X,Y) = P(X)P(Y) for ALL values Cov(X,Y) = 0 → ρ = 0
Strength Stronger condition — rules out ALL relationships (linear and non-linear) Weaker condition — rules out only LINEAR relationships; non-linear relationships can still exist
Implies the other? Independence → uncorrelated (always) Uncorrelated does NOT → independent
E(XY) rule E(XY) = E(X) × E(Y) E(XY) = E(X) × E(Y) [same rule applies]
★ Expected Value of Product — Key Rule
If X and Y are uncorrelated (or independent):
E(XY) = E(X) × E(Y)

Derivation: E(XY) = E(X)E(Y) + Cov(X,Y)
When Cov(X,Y) = 0: E(XY) = E(X)E(Y)
Useful for financial variables that are products of random quantities (e.g., Revenue = Price × Quantity).
If price and quantity are uncorrelated, E(Revenue) = E(Price) × E(Quantity).
⚠️ Critical Exam Distinction — Independence vs. Uncorrelated

Independence is STRONGER than uncorrelatedness. A variable can be uncorrelated with another (ρ = 0) yet still NOT be independent — there could be a non-linear relationship.

Example: Y = X² where X ~ symmetric around 0. Cov(X, Y) = 0 (uncorrelated), but Y is completely determined by X (not independent).

On the exam: if told "independent," you can conclude uncorrelated. If told "uncorrelated," you CANNOT conclude independent.

09

Shortfall Risk & Roy's Safety-First Criterion

Context: Mean-variance analysis (MPT) treats risk symmetrically (standard deviation captures both upside and downside variability). Safety-first rules focus only on downside risk — specifically the risk of falling below a minimum acceptable return level R_L.

Mean-variance analysis holds exactly when EITHER:

  1. Returns are normally distributed, OR
  2. Investors have quadratic utility functions

It holds approximately even when these assumptions are violated — which is why it remains widely used.

★ Safety-First Ratio (SFRatio)
SFRatio = [E(Rp) − R_L] / σp

Where:
E(Rp) = expected portfolio return
R_L = minimum acceptable threshold return (shortfall level)
σp = portfolio standard deviation

Probability of falling below R_L:
P(Rp < R_L) = N(−SFRatio)
Numerator [E(Rp) − R_L] = distance from expected return to the shortfall level.
Dividing by σp converts this distance to standard deviation units.
Higher SFRatio → greater distance from the threshold in std dev units → lower probability of shortfall.
N(−SFRatio) = 1 − N(SFRatio) using standard normal table symmetry.
Special case: When R_L = Rf (risk-free rate), SFRatio = Sharpe Ratio.

Roy's Safety-First Criterion — Decision Rule

Roy's safety-first criterion: The optimal portfolio minimizes the probability that portfolio return falls below R_L.

Assuming normally distributed returns, this is equivalent to choosing the portfolio with the HIGHEST SFRatio.

Two-step implementation:

  1. Calculate each portfolio's SFRatio = [E(Rp) − R_L] / σp
  2. Choose the portfolio with the highest SFRatio → lowest shortfall probability
📊 Worked Example — Roy's Safety-First Criterion

Threshold R_L = 2%. Two portfolios to compare:

Portfolio 1: E(R) = 12%, σ = 15%
Portfolio 2: E(R) = 14%, σ = 16%

SFRatio₁ = (12% − 2%) / 15% = 10/15 = 0.667
SFRatio₂ = (14% − 2%) / 16% = 12/16 = 0.750

Portfolio 2 has higher SFRatio → select Portfolio 2

Probability of shortfall for Portfolio 2:
P(Rp < 2%) = N(−0.75) = 1 − N(0.75) = 1 − 0.7734 = 0.2266 ≈ 22.7%
→ Optimal: Portfolio 2 (SFRatio = 0.75, shortfall probability ≈ 22.7%)
💡 SFRatio = Sharpe Ratio when R_L = Risk-Free Rate

Sharpe Ratio = [E(Rp) − Rf] / σp
SFRatio = [E(Rp) − R_L] / σp

When R_L = Rf: SFRatio = Sharpe Ratio exactly.

Interpretation of Sharpe Ratio via safety-first: choosing the portfolio with the highest Sharpe Ratio minimizes the probability that portfolio return will fall below the risk-free rate (assuming normally distributed returns). This gives the Sharpe Ratio a new, intuitive downside-risk interpretation.

💡 Link to Value at Risk (VaR)

VaR is also an application of the normal distribution to portfolio risk. VaR = minimum value of losses expected over a specified time period at a given probability level. The variance-covariance (analytical) method for estimating VaR assumes normally distributed returns — the same assumption underlying Roy's safety-first criterion and mean-variance analysis. Stress testing and scenario analysis complement VaR for tail-risk estimation.

⚠️ Key Exam Traps — Safety-First

1. Higher SFRatio = LOWER shortfall probability (not higher). Always select the portfolio with the HIGHEST SFRatio.

2. P(Rp < R_L) = N(−SFRatio) — the argument is NEGATIVE SFRatio, not positive. Use standard normal table: N(−z) = 1 − N(z).

3. SFRatio requires normality assumption. Without normally distributed returns, this approach does not apply exactly.

4. SFRatio ≠ Sharpe Ratio unless R_L = Rf specifically.

Exam Checklist — Must-Know Points

  • ✅ E(Rp) = Σ wᵢ · E(Rᵢ) — simple weighted average of individual expected returns. Weights must sum to 1. This IS a weighted average (unlike variance).
  • ✅ Portfolio variance is NOT a weighted average of individual variances — it also includes covariance (cross-product) terms between every pair of assets.
  • ✅ 2-asset variance: σ²(Rp) = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(R₁,R₂). The covariance term has a factor of 2 because Cov(1,2) = Cov(2,1).
  • ✅ Cov = ρ × σᵢ × σⱼ — the rearrangement of the correlation formula; plug this directly into the 2-asset variance formula when given ρ and individual σ values.
  • ✅ Covariance: unbounded, has units (e.g., %²), tells direction of co-movement. Sign: negative = inverse relationship, 0 = unrelated, positive = moves together.
  • ✅ Correlation: bounded [−1, +1], dimensionless, tells direction AND standardized strength. ρ = +1: no diversification benefit. ρ < +1: portfolio σ < weighted-average σ (diversification works).
  • ✅ Cov(R, R) = σ²(R) — own covariance = own variance. Diagonal of covariance matrix = variances. Diagonal of correlation matrix = 1.0 always.
  • ✅ Covariance matrix is symmetric: Cov(Rᵢ,Rⱼ) = Cov(Rⱼ,Rᵢ). n assets → n variance terms + n(n−1) covariance terms. As n grows, covariances dominate.
  • ✅ General portfolio variance: σ²(Rp) = Σᵢ Σⱼ wᵢwⱼCov(Rᵢ,Rⱼ) — works for any number of assets; diagonal terms are variances, off-diagonal are covariances.
  • ✅ Covariance from joint probability: Cov(Rₐ,R_b) = Σᵢ Σⱼ P(Rₐ,ᵢ, R_b,ⱼ)(Rₐ,ᵢ−ERₐ)(R_b,ⱼ−ER_b). Sum all probability-weighted cross-products of deviations.
  • ✅ Independence: P(X,Y) = P(X)P(Y) — stronger than uncorrelated. Independence → uncorrelated. Uncorrelated does NOT imply independent.
  • ✅ E(XY) = E(X)E(Y) when X and Y are uncorrelated (since E(XY) = E(X)E(Y) + Cov(X,Y), and Cov = 0). Also holds for independent variables.
  • ✅ Shortfall risk = risk that Rp falls below minimum acceptable level R_L. Safety-first rules focus only on downside risk (unlike std dev which is symmetric).
  • ✅ SFRatio = [E(Rp) − R_L] / σp. Higher SFRatio = lower shortfall probability. Roy's criterion: select the portfolio with the HIGHEST SFRatio.
  • ✅ P(Rp < R_L) = N(−SFRatio) — uses standard normal table with NEGATIVE SFRatio as the argument. N(−z) = 1 − N(z).
  • ✅ When R_L = Rf: SFRatio = Sharpe Ratio. Sharpe Ratio maximization ≡ minimizing probability of underperforming the risk-free rate (under normality).
  • ✅ Mean-variance analysis requires either (1) normally distributed returns OR (2) quadratic utility functions. Holds approximately even when violated.
  • ✅ VaR connection: VaR = minimum value of losses at a given probability. The analytical (variance-covariance) VaR method assumes normally distributed returns — same assumption as Roy's criterion.
  • ✅ Diversification benefit increases as correlation DECREASES. At ρ = +1: no benefit (σp = weighted-average σ). At ρ = −1: maximum benefit (can reach σp = 0).

Cov(Rₐ,R_b) = Σᵢ Σⱼ P(Rₐ,ᵢ, R_b,ⱼ)(Rₐ,ᵢ−ERₐ)(R_b,ⱼ−ER_b)

═══ INDEPENDENCE / UNCORRELATED ═══
Independent: P(X,Y) = P(X)P(Y)           [stronger condition]
E(XY) = E(X)E(Y) if uncorrelated         [since E(XY) = E(X)E(Y) + Cov(X,Y)]

═══ SAFETY-FIRST ═══
SFRatio = [E(Rp) − R_L] / σp             [maximize to minimize shortfall risk]
P(Rp < R_L) = N(−SFRatio)               [normal distribution; argument is NEGATIVE]
Sharpe Ratio = [E(Rp) − Rf] / σp         [= SFRatio when R_L = Rf]
Roy's criterion: SELECT portfolio with HIGHEST SFRatio
E(Rp) IS a weighted average; σ²(Rp) is NOT. Variance requires covariance terms between every pair.
Covariance: unbounded, has units. Correlation: bounded [−1,+1], dimensionless. Cov = ρ × σᵢ × σⱼ is the key link.
As number of assets grows: covariance terms (n²−n) dominate variance terms (n). For 20 assets: 380 covariances vs. 20 variances.
Independence → uncorrelated. Uncorrelated ≠ independent (non-linear relationships can still exist).
Roy's criterion: HIGHEST SFRatio = LOWEST shortfall probability. P(Rp < R_L) = N(−SFRatio) — note NEGATIVE sign.
SFRatio = Sharpe Ratio only when threshold R_L = risk-free rate Rf.
