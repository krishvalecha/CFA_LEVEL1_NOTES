01

The Big Idea — Four Building Blocks, One Relationship

Put–call parity is one of the most powerful and testable relationships in all of derivatives. It links four instruments — a call option, a put option, the underlying asset, and a risk-free bond — into a single no-arbitrage equation. Knowing any three, you can derive the fourth. And by rearranging the equation, you can replicate any one of the four instruments using the other three.

The core logic: two portfolios that produce identical cash flows under all scenarios at time T must have the same price at time 0. If they don't, a riskless arbitrage profit is available.

📋 Call Option (c₀)

Right to BUY underlying at X. Payoff: Max(0, S_T − X). Costs c₀ upfront. Gains when S rises.

📋 Put Option (p₀)

Right to SELL underlying at X. Payoff: Max(0, X − S_T). Costs p₀ upfront. Gains when S falls.

📋 Risk-Free Bond (PV(X))

Costs X(1+r)^−T today. Returns exactly X at time T. No default risk. The "floor" that protects value.

💡 Why This Module Matters

Put–call parity lets you: (1) price a put if you know the call (and vice versa); (2) identify and exploit mispricing; (3) understand how standard investment strategies (covered call, protective put) can be replicated; (4) extend to forward contracts; (5) model the financial claims of shareholders and debtholders using option theory. Every CFA exam tests at least one of these applications.

Protective Put vs. Fiduciary Call — Identical Payoffs, Same Price

Portfolio 2: Protective Put

What it is: Long underlying (S₀) + Long put option (p₀)

Cost at t=0: S₀ + p₀

Goal: Participate in upside of the underlying; the put provides a floor — if S_T < X, the put kicks in and you receive X regardless.

Analogy: Owning a stock and buying insurance against it falling below X.

Portfolio 1: Fiduciary Call

What it is: Long call option (c₀) + Long risk-free bond that pays X at T [costs X(1+r)^−T today]

Cost at t=0: c₀ + X(1+r)^−T

Goal: The bond guarantees X at T (downside floor). The call captures any upside above X.

Why "fiduciary": The bond ensures you always have X available to exercise the call — responsible/fiduciary management of capital.

★ The Key Insight — Identical Payoffs in ALL Scenarios

Both portfolios produce exactly the same payoff in every scenario:

  • When S_T < X: Both pay X (put protects / bond provides X)

No-arbitrage condition: Identical future cash flows → identical current prices. Therefore: S₀ + p₀ = c₀ + X(1+r)^−T

03

Put–Call Parity — Equation 1 & All Rearrangements

Equation 1 — Put–Call Parity (the Master Equation)
S₀ + p₀ = c₀ + X(1+r)^−T

[Protective Put] = [Fiduciary Call]

Where:
S₀ = current underlying price
p₀ = put option premium (same X and T as call)
c₀ = call option premium
X = exercise price (same for both options)
r = risk-free rate (per period)
T = time to expiration
X(1+r)^−T = PV of the exercise price = cost of the risk-free bond today
Critical requirements for put–call parity to hold:
(1) Both options are EUROPEAN (exercisable only at T)
(2) Same underlying, same exercise price X, same expiration T
(3) No dividends or income on the underlying (covered in the parity extension for assets with income)
(4) Can borrow and lend at the risk-free rate

Not just a formula — it's a no-arbitrage identity. If LHS ≠ RHS, you can earn riskless profit.

All Four Rearrangements — Solving for Each Component

Equation 2 — Solving for Put (p₀)
p₀ = c₀ + X(1+r)^−T − S₀

Replication: Long call + Long bond + Short underlying = Long put
The put can be created synthetically without ever buying a put directly.
Equation — Solving for Call (c₀)
c₀ = S₀ + p₀ − X(1+r)^−T

Replication: Long underlying + Long put + Short bond = Long call
A long call can be replicated: own the asset, insure it with a put, borrow PV(X).
Equation 3 — Solving for Underlying (S₀)
S₀ = c₀ − p₀ + X(1+r)^−T

Replication: Long call + Short put + Long bond = Long underlying
The asymmetric payoffs of the call and put offset each other,
leaving a linear (underlying-like) payoff.
Equation — Solving for Risk-Free Bond (PV(X))
X(1+r)^−T = S₀ + p₀ − c₀

Replication: Long underlying + Long put + Short call = Long risk-free bond
This is the key insight: a covered call position plus a put = a risk-free bond.

Worked Example — Biomian Put–Call Parity (Example 1)

VFO: Biomian shares S₀ = INR295 | X = INR265 | T = 0.5 years | r = 4% | c₀ = INR59
Step 1: Calculate PV(X) = 265 × (1.04)^−0.5 = 265 × 0.9806 = INR259.85

Step 2: Apply put–call parity (Eq. 1):
S₀ + p₀ = c₀ + PV(X)
295 + p₀ = 59 + 259.85
295 + p₀ = 318.85
p₀ = INR23.85

Check: Protective Put cost = 295 + 23.85 = INR318.85 ✓
Fiduciary Call cost = 59 + 259.85 = INR318.85 ✓
Both portfolios cost the same → no-arbitrage holds.
04

Arbitrage When Put–Call Parity is Violated — Example 2

If put–call parity is violated (i.e., the two portfolios trade at different prices), a riskless arbitrage profit is available. The strategy: sell the overpriced portfolio, buy the underpriced portfolio. The combined position will have zero cash flows at maturity under every scenario, locking in the price difference as a riskless profit at t=0.

⚠️ Identifying the Arbitrage — Always Compare Both Sides

Step 1: Check if S₀ + p₀ = c₀ + X(1+r)^−T
Step 2: If LHS > RHS → Protective put is overpriced → SELL protective put (sell the stock + sell the put), BUY fiduciary call (buy the call + buy the bond)
Step 3: If LHS < RHS → Fiduciary call is overpriced → SELL fiduciary call (sell the call + sell the bond), BUY protective put (buy the stock + buy the put)
Step 4: Profit = |LHS − RHS| — collected at t=0 with zero net cash flows at T.

Worked Arbitrage Example (Example 2)

Same as Example 1 but: observed put price = INR30 (fair value = INR23.85)
Check parity: S₀ + p₀ vs. c₀ + PV(X)
295 + 30 = 325 vs. 59 + 259.85 = 318.85
325 > 318.85 → PROTECTIVE PUT IS OVERPRICED by INR6.15

ARBITRAGE STRATEGY:
At t=0: Sell the protective put (sell stock + sell put) → receive 295 + 30 = +325
Buy the fiduciary call (buy call + buy bond) → pay 59 + 259.85 = −318.85
NET CASH INFLOW at t=0: +INR6.15 ← this is the riskless profit

💡 The Perfect Arbitrage — Why Zero Net Cash Flows at T

The short protective put and long fiduciary call produce exactly offsetting cash flows at T in every scenario. This means the INR6.15 received at t=0 is pure riskless profit — you've locked in a gain with zero net exposure going forward. This is the textbook definition of arbitrage: positive cash flow today, zero net cash flows tomorrow, no risk. Market forces will push the put price back toward INR23.85 as arbitrageurs exploit this.

Replication of Individual Positions — Exhibit 6 (Master Table)

Put–call parity can be rearranged to show how each of the four building blocks can be replicated using the other three. This is the most exam-tested part of this module. You must know which three instruments replicate each position — and what sign (long or short) each component takes.

To Replicate This Position → Use Underlying (S₀) Use Risk-Free Bond (PV(X)) Use Call (c₀) Use Put (p₀)
Long Underlying (S₀) — Long Long Short
Long Risk-Free Bond (PV(X)) Long — Short Long
Long Call (c₀) Long Short — Long
Long Put (p₀) Short Long Long —
💡 How to Read This Table — and Derive It Yourself

Start with Eq. 1: S₀ + p₀ = c₀ + X(1+r)^−T. To replicate any position, move it to the right side and rearrange. Whatever is on the right = the replicating portfolio. Long = positive sign; Short = negative sign.

Example — Replicate Long Put: p₀ = c₀ + X(1+r)^−T − S₀ → Long call + Long bond + Short underlying ✓
Example — Replicate Long Call: c₀ = S₀ + p₀ − X(1+r)^−T → Long underlying + Long put + Short bond ✓
Example — Replicate Long Bond: X(1+r)^−T = S₀ + p₀ − c₀ → Long underlying + Long put + Short call ✓
Example — Replicate Long Underlying: S₀ = c₀ − p₀ + X(1+r)^−T → Long call + Short put + Long bond ✓

⚠️ Exam Trap — Short Positions Are Negatives in the Equation

When you "short" an instrument in a replicating portfolio, it means you SELL it — you receive cash now and have an obligation at T. In the put–call parity equation, short = negative sign. So "short bond" means you borrow PV(X) today and repay X at T. "Short underlying" means you sell the asset now and must buy it back at T. Always track the signs carefully in both the cost at t=0 and the cash flows at T.

Option Strategies via Put–Call Parity — Covered Call & Protective Put

Strategy 1: Covered Call

What it is: Long underlying + Short call. Written as: S₀ − c₀.

Who uses it: An investor who owns the stock and believes it will be flat in the near term. They sell a call to generate income (the premium). If S_T > X, they must sell the stock at X (forgoing further upside). If S_T ≤ X, they keep the stock and the premium.

Payoff: If S_T < X: S_T (keep stock). If S_T > X: X (stock sold at X, giving up upside above X).

Covered Call Replication via Put–Call Parity
Start: S₀ + p₀ = c₀ + X(1+r)^−T
Rearrange for (S₀ − c₀):
S₀ − c₀ = X(1+r)^−T − p₀

COVERED CALL = Long risk-free bond + Short put option

Intuition:
• The bond provides the X floor (you always receive at least X in PV terms)
• The short put obligates you to buy the stock at X if S_T < X (capping your downside just like a sold call combined with owning the stock)
• Both positions cap upside at X and have downside through the short put / sold call
Payoff check — covered call vs. short put + bond:
If S_T < X: Covered call pays S_T. Short put + bond pays X − (X − S_T) = S_T ✓
If S_T > X: Covered call pays X. Short put + bond pays 0 + X = X ✓
Identical payoffs → identical prices.

Worked Example — VFO Covered Call (Example 3)

S₀ = INR295 | X = INR325 | T = 0.5yr | r = 4% | p₀ = INR56 | Find c₀
Use: S₀ − c₀ = X(1+r)^−T − p₀
PV(X) = 325 × (1.04)^−0.5 = 325 × 0.9806 = INR318.69
295 − c₀ = 318.69 − 56
295 − c₀ = 262.69
c₀ = 295 − 262.69 = INR32.31

The no-arbitrage call premium is INR32.31.
Verify: S₀ + p₀ = 295 + 56 = 351 | c₀ + PV(X) = 32.31 + 318.69 = 351 ✓

Strategy 2: Protective Put (Recap)

What it is: Long underlying + Long put. Written as: S₀ + p₀.

Who uses it: An investor who wants upside participation in the stock but fears a drop. The put provides a floor: if S_T < X, the put pays X − S_T so the total portfolio value is always at least X.

Replication: From Eq. 1: S₀ + p₀ = c₀ + X(1+r)^−T. The protective put equals a fiduciary call. Alternatively: S₀ + p₀ = long underlying + long put = long call + long bond.

Put–Call Forward Parity — Equations 4 & 5

Put–call forward parity extends the basic put–call parity by replacing the cash underlying (S₀) with a synthetic underlying position — a long forward contract plus a risk-free bond. Since we already know that: Long Forward + Long Bond (PV(F₀(T))) = Long Underlying, we substitute this into Eq. 1.

Equation 4 — Put–Call Forward Parity
Recall: S₀ = F₀(T) × (1+r)^−T + 0 [long forward costs nothing; bond costs PV(F₀(T))]
Substitute S₀ = F₀(T)(1+r)^−T into Eq. 1:

F₀(T)(1+r)^−T + p₀ = c₀ + X(1+r)^−T

[Synthetic Protective Put] = [Fiduciary Call]

Where the synthetic protective put = Long forward + Long bond (PV(F₀(T))) + Long put
Synthetic protective put: Instead of buying the underlying directly, you replicate ownership using a forward + a bond (PV of forward price). Adding a put on top gives the same payoff as a traditional protective put.

Why use this? Forwards are often more liquid and capital-efficient than holding the physical underlying. A fund manager can use a futures contract instead of buying shares, while still using put–call parity to price the put or call.
Equation 5 — Key Rearrangement: Long Put / Short Call = Long Bond + Short Forward
From Eq. 4: F₀(T)(1+r)^−T + p₀ = c₀ + X(1+r)^−T
Rearrange:
p₀ − c₀ = X(1+r)^−T − F₀(T)(1+r)^−T
p₀ − c₀ = [X − F₀(T)](1+r)^−T

MEANING:
Long put + Short call = Long bond (PV of X) + Short forward (PV of F₀(T))

When X = F₀(T) (at-the-money forward options):
p₀ = c₀ — ATM call and ATM put have the same price!
If X > F₀(T): p₀ > c₀ (put is more valuable — you can sell at X which is above the forward price)
If X < F₀(T): c₀ > p₀ (call is more valuable — you can buy at X which is below the forward price)
If X = F₀(T): p₀ = c₀ (both options equally likely to be exercised → same price)

💡 When X = F₀(T): ATM Call = ATM Put

When the exercise price equals the forward price, the option is "at the money forward." In this case: p₀ − c₀ = [X − F₀(T)](1+r)^−T = 0 → p₀ = c₀. This is an important exam fact: for at-the-money forward European options, the call and put have the same price. This is intuitive — both options are equally likely to be exercised since X = the expected future price.

Firm Value Application — Equity & Debt as Options (Equation 6)

One of the most elegant applications of put–call parity is in corporate finance: the financial claims of shareholders and debtholders can be described as option positions on the value of the firm. This insight connects derivatives theory to credit risk and capital structure — a crossover topic the exam tests.

The Setup
A firm at t=0 has asset value V₀ = E₀ + PV(D), where E₀ = equity value and PV(D) = present value of zero-coupon debt with face value D maturing at T. At T, the firm's assets (V_T) are distributed to shareholders and debtholders based on whether the firm is solvent or insolvent.
✅ Solvency (V_T > D)
  • Firm can repay debt in full
  • Debtholders receive: D (repaid in full)
  • Shareholders receive: V_T − D (residual after debt)
  • Shareholder payoff = Max(0, V_T − D) when solvent → V_T − D > 0
⚠️ Insolvency (V_T < D)
  • Firm cannot repay debt — defaults
  • Debtholders receive: V_T (all assets, still less than D owed)
  • Shareholders receive: 0 (equity wiped out)
  • Shareholder payoff = Max(0, V_T − D) when insolvent → 0

Shareholders and Debtholders as Option Positions

Shareholder Payoff = Long Call on Firm Value
Shareholder payoff at T = Max(0, V_T − D)

This is IDENTICAL to a long call option on the firm's assets (V_T)
with exercise price = D (the debt face value).

Intuition:
• If V_T > D: Shareholders exercise — pay off debt D, keep residual (V_T − D)
• If V_T < D: Shareholders "don't exercise" — let firm default, walk away with 0
• Unlimited upside (if firm grows, equity grows). Downside capped at 0 (limited liability).

Equity = Long Call on Firm Value (exercise price = D)
Debtholder Payoff = Long Risk-Free Bond + Short Put on Firm Value
Debtholder payoff at T = Min(V_T, D)

This equals: D − Max(0, D − V_T)
= Long risk-free bond (D) + Short put on V_T with exercise price D

Intuition:
• If V_T > D (solvent): Receive D (bond pays off). Short put expires worthless → net = D
• If V_T < D (insolvent): Receive D but pay (D − V_T) on the short put → net = V_T
• Limited upside (at most D, even if firm grows dramatically)
• Downside risk: if firm defaults, receive less than D owed

Risky Debt = Long Risk-Free Bond − Put Option on Firm Value
Equation 6 — Firm Value via Put–Call Parity
Apply put–call parity: S₀ + p₀ = c₀ + PV(X)
Substitute: V₀ for S₀, D for X, E₀ for c₀, risky debt position for p₀:

V₀ + p₀ = c₀ + PV(D)
V₀ = c₀ + PV(D) − p₀

WHERE:
V₀ = total firm asset value at t=0
c₀ = equity value (call option on firm assets, exercise price = D)
PV(D) = present value of risk-free debt
p₀ = put option on firm value = CREDIT SPREAD on the debt

Interpretation:
Firm value = Equity (call) + Risk-free debt − Put option (credit spread)
OR: Firm value = Equity value + Risky debt value
The put option (p₀) as credit spread: The put represents the premium above the risk-free rate that the firm must pay to debtholders to compensate them for assuming insolvency risk. The more likely insolvency (lower V₀ relative to D), the more valuable p₀, the higher the credit spread.

This is the foundation of structural credit models (e.g., Merton model) used in credit risk assessment.

Summary — Firm Value Option Analogy

⚠️ Exam Trap — Shareholders Have LIMITED Downside, NOT Unlimited

Students sometimes confuse the analogy. Shareholders are like CALL BUYERS — they have unlimited upside (if firm value soars) and limited downside (equity can only go to zero, not negative — limited liability). Debtholders are like CALL SELLERS / BOND HOLDERS + SHORT PUT — they have limited upside (maximum D) and bear downside through the short put (they absorb losses when V_T < D). The asymmetry between shareholders and debtholders is precisely the put option embedded in risky debt.

💡 Credit Spread Intuition — The Put Option Connection

The credit spread on a bond (the yield above the risk-free rate) compensates debtholders for the insolvency risk they bear (the sold put). When a company is financially distressed (V₀ close to D), the put option is deep in the money → very valuable → large credit spread. When a company is financially healthy (V₀ much greater than D), the put is deep out of the money → nearly worthless → small credit spread. This is why credit spreads widen when firms' asset values decline relative to their debt obligations.

Exam Checklist & Quick Reference Summary

  • ✅ Put–Call Parity (Eq. 1): S₀ + p₀ = c₀ + X(1+r)^−T. Applies to European options with same X and same T on the same underlying (no income/costs).
  • ✅ Protective Put = Fiduciary Call: Both produce Max(X, S_T) under all scenarios. This equivalence IS put–call parity.
  • ✅ Fiduciary call: Long call + Long bond (pays X at T). Costs c₀ + X(1+r)^−T.
  • ✅ Protective put: Long underlying + Long put. Costs S₀ + p₀.
  • ✅ Solving for put: p₀ = c₀ + X(1+r)^−T − S₀. To get the put: buy a call, buy a bond, short the underlying.
  • ✅ Solving for call: c₀ = S₀ + p₀ − X(1+r)^−T. To get the call: buy the stock, buy a put, short the bond (borrow PV(X)).
  • ✅ Solving for underlying: S₀ = c₀ − p₀ + X(1+r)^−T. Synthetic underlying: long call + short put + long bond.
  • ✅ Solving for bond: X(1+r)^−T = S₀ + p₀ − c₀. Synthetic bond: long underlying + long put + short call (also known as a conversion).
  • ✅ Arbitrage rule: If S₀ + p₀ > c₀ + PV(X) → sell protective put, buy fiduciary call. Riskless profit = difference. Net cash flows at T = 0 under ALL scenarios.
  • ✅ Covered call = Long bond + Short put: From S₀ − c₀ = X(1+r)^−T − p₀. The covered call has capped upside (at X) and downside through the short put / falling stock.
  • ✅ Put–Call Forward Parity (Eq. 4): F₀(T)(1+r)^−T + p₀ = c₀ + X(1+r)^−T. Substitute PV(F₀(T)) for S₀ in the basic parity equation.
  • ✅ Key rearrangement (Eq. 5): p₀ − c₀ = [X − F₀(T)](1+r)^−T. Long put + Short call = Long bond (PV(X)) + Short forward (PV(F₀(T))).
  • ✅ ATM Forward options: p₀ = c₀. When X = F₀(T), both options have the same price. When X > F₀(T), put > call. When X < F₀(T), call > put.
  • ✅ Synthetic protective put: Long forward + Long bond (PV(F₀(T))) + Long put. Identical payoff to cash protective put and to fiduciary call.
  • ✅ Shareholder payoff = Max(0, V_T − D) = Long Call on firm assets with exercise price D. Unlimited upside; downside capped at 0 (limited liability).
  • ✅ Debtholder payoff = Min(V_T, D) = Long Bond + Short Put on firm assets with exercise price D. Upside capped at D; downside exposed to insolvency.
  • ✅ Firm value equation (Eq. 6): V₀ = c₀ + PV(D) − p₀. Firm value = Equity (call) + PV(risk-free debt) − Credit spread (put).
  • ✅ The put (p₀) in firm value = credit spread: It's the premium debtholders demand above the risk-free rate to compensate for insolvency risk. Higher insolvency risk → higher p₀ → wider credit spread.
  • ✅ As firm approaches distress: p₀ (the insolvency put) becomes more valuable. Shareholders benefit from this (they're long the put in a way — they "put" the firm to debtholders in insolvency). Debtholders bear more risk (they're short the put).

Solve for p₀: p₀ = c₀ + X(1+r)^−T − S₀
Solve for c₀: c₀ = S₀ + p₀ − X(1+r)^−T
Solve for S₀: S₀ = c₀ − p₀ + X(1+r)^−T
Solve for bond: X(1+r)^−T = S₀ + p₀ − c₀
Covered call: S₀ − c₀ = X(1+r)^−T − p₀

═══ REPLICATION (Exhibit 6) ═══
Long underlying = Long bond + Long call + Short put
Long bond = Long S + Long p + Short c
Long call = Long S + Long p + Short bond
Long put = Short S + Long bond + Long call

═══ PUT–CALL FORWARD PARITY ═══
F₀(T)(1+r)^−T + p₀ = c₀ + X(1+r)^−T [Eq. 4]
p₀ − c₀ = [X − F₀(T)](1+r)^−T [Eq. 5]
When X = F₀(T): p₀ = c₀ (ATM forward → same price)

═══ FIRM VALUE APPLICATION ═══
Shareholder payoff = Max(0, V_T − D) = Long call (X=D)
Debtholder payoff = Min(V_T, D) = Long bond + Short put
Firm value: V₀ = c₀ + PV(D) − p₀ [Eq. 6]
p₀ = credit spread (insolvency risk premium)

═══ ARBITRAGE RULE ═══
LHS > RHS → Sell protective put, Buy fiduciary call
LHS < RHS → Sell fiduciary call, Buy protective put
Profit = |S₀ + p₀ − c₀ − X(1+r)^−T| collected at t=0
Put–call parity requires: European options only (not American); same X and same T for both options; same underlying; no dividends/income; can borrow/lend at r. Any violation = potential arbitrage.

Covered call = short put + long bond — not intuitively obvious but follows directly from parity. The key insight: selling a call caps your upside; a short put has the same downside shape; combining with a bond provides the X floor.

Firm value application: The most conceptually deep part of this module. Connect it to credit risk: equity = call, risky debt = bond − put, credit spread = put premium. More risk → more valuable put → wider spread.
