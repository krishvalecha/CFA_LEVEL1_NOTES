Interest Rates — Three Interpretations

Interest Rate (r)
A rate of return that reflects the relationship between differently dated (timed) cash flows. It establishes equivalence between cash flows occurring at different points in time. Example: if USD 9,500 today = USD 10,000 in one year, then r = USD500 / USD9,500 = 5.26%.
1 — Required Rate of Return

The minimum rate of return an investor must receive to accept an investment. Compensates for time value of money and risk.

2 — Discount Rate

The rate used to find the present value of a future cash flow. "What rate makes future cash flows equivalent to a present amount?"

"Interest rate" and "discount rate" are used almost interchangeably.

3 — Opportunity Cost

The value forgone by choosing one course of action over another. If you spend USD 9,500 today instead of saving it, you forgo the 5.26% return.

💡 Key Concept — All Three Are the Same Rate

Whether we call it required rate of return, discount rate, or opportunity cost — it's the same number, just viewed from three different angles. The rate always embeds the real risk-free rate + applicable risk premia.

02

Interest Rate Components — The Building Blocks

Interest Rate Decomposition (The Master Formula)
r = Real risk-free rate + Inflation premium + Default risk premium + Liquidity premium + Maturity premium

Nominal risk-free rate (exact): (1 + nominal) = (1 + real risk-free)(1 + inflation premium)
Nominal risk-free rate (approx): Nominal ≈ Real risk-free rate + Inflation premium
Mnemonic: "RIDLM" — Real · Inflation · Default · Liquidity · Maturity
Or: "Real Investors Demand Large Margins"
Real Risk-Free Rate

Single-period rate for a completely risk-free security with zero inflation. Reflects time preferences for current vs. future real consumption.

Inflation Premium

Compensates for expected inflation over the maturity of the debt. Reflects the average expected inflation rate. Inflation erodes purchasing power.

Default Risk Premium

Compensates for the possibility that the borrower will fail to make a promised payment at the contracted time and amount.

Liquidity Premium

Compensates for the risk of loss vs. fair value if the investment must be converted to cash quickly. US T-bills have NO liquidity premium (highly liquid). Small issuer bonds have a liquidity premium.

Maturity Premium

Compensates for the increased price sensitivity of longer-maturity debt to changes in market interest rates. Longer maturity = greater price volatility = higher maturity premium.

Nominal risk-free rate in practice: Short-term government debt interest rates represent the nominal risk-free rate. Examples:

  • USA: 90-day T-bill rate = nominal risk-free rate for the US over the next 3 months
  • France: BTFs (Bons du Trésor) with maturities up to one year
  • Japan: Short-term Treasury bills with maturities of 6 and 12 months

Important: Interest rates quoted at 3% for a 90-day security are annualized rates — not the actual rate earned over 90 days.

All premia vary continuously over time with changes in economic fundamentals and expectations. All interest rates therefore fluctuate.

⚠️ Exact vs. Approximate Nominal Rate

The exact formula is: (1 + nominal) = (1 + real)(1 + inflation). The approximation is: nominal ≈ real + inflation. The approximation underestimates slightly because it ignores the cross-product term (real × inflation). For the exam, know both and use exact unless told to approximate.

03

Holding Period Return (HPR)

Holding Period Return (R)
The return earned from holding an asset for a single specified period of time — one day, one week, one month, five years, or any period. Total return = capital gain/loss + income yield.
Holding Period Return Formulas
Single Period: R = (P₁ − P₀ + I₁) / P₀

Where: P₀ = beginning price, P₁ = ending price, I₁ = income (dividends/interest) received

Multi-Period (compounded over T periods):
R = [(1 + R₁) × (1 + R₂) × ... × (1 + Rₜ)] − 1
Two components of total return:
• Capital gain component: (P₁ − P₀) / P₀
• Income yield component: I₁ / P₀
Total return = capital gain % + income yield % (only if income received at end of period; if received earlier and reinvested, total return could be higher)
💡 Return Notation

Returns can be expressed equivalently as: decimals (0.07), fractions (7/100), or percentages (7%). For multi-period compounding, always use the (1 + R) format — never just multiply returns directly.

04

Five Types of Mean Returns

1 — Arithmetic Mean Return (R̄)

Simple average of holding period returns. Easiest to compute; has well-known statistical properties.

R̄ᵢ = (R₁ + R₂ + ... + Rₜ) / T = (1/T) × Σ Rᵢₜ
  • Assumes the same amount is invested at the beginning of each period
  • Always ≥ geometric mean (equal only if all returns are identical)
  • Biased upward unless all returns are equal; bias is especially severe with mixed positive and negative returns
  • Best for estimating the average return over a single future period
  • Example: Returns of −50%, +35%, +27% → Arithmetic mean = (−50+35+27)/3 = 4%
2 — Geometric Mean Return (R̄G)

Compound rate of growth — the "true" multi-period return.

R̄Gᵢ = [(1+R₁)(1+R₂)...(1+Rₜ)]^(1/T) − 1
  • Accounts for compounding — base changes each period
  • More accurate representation of portfolio growth over multiple periods
  • Example: Returns of −50%, +35%, +27% → Geometric mean = [(0.5)(1.35)(1.27)]^(1/3) − 1 = −5.00%
  • EUR1 invested → EUR0.8574 after 3 years (matches actual). Using arithmetic mean (4%) → EUR1.1249 (wrong!)
  • Best for estimating average returns over multiple future periods and reporting historical performance
  • Difference between arithmetic and geometric means increases with greater variability in the data
3 — Harmonic Mean Return (X̄H)

Reciprocal-based mean. Used for averaging rates and ratios; less affected by large outliers.

X̄H = n / Σ(1/Xᵢ) [where Xᵢ > 0]
  • Weight of each observation is inversely proportional to its magnitude → reduces influence of large values
  • Used most often when data consist of rates and ratios (e.g., P/E ratios)
  • Classic application: cost averaging — periodic investment of a fixed dollar amount (average cost per share)
  • Example: P/Es of 45, 15, 15 → Arithmetic mean = 25, Harmonic mean = 19.3 (lower; down-weights the outlier 45)
  • For negative returns: convert to (1+R), compute harmonic mean, subtract 1
  • Only works for non-negative numbers
4 — Trimmed Mean

Removes a defined percentage of the largest and smallest values before calculating the arithmetic mean. Minimizes impact of extreme outliers but discards them entirely.

  • Example: "5% trimmed mean" = remove the top 5% and bottom 5% of observations, then average the rest
  • Use when: there are extreme outliers AND you want to exclude them from the analysis
5 — Winsorized Mean

Replaces extreme observations with the values of their nearest non-extreme observations, then takes the arithmetic mean. Limits but does not discard outliers.

  • Example: In a dataset, the top and bottom 5% of observations are replaced by the 95th and 5th percentile values respectively, then averaged
  • Use when: there are extreme outliers AND you want to include but dampen their impact
  • Key difference from trimmed mean: winsorized replaces; trimmed removes
05

Which Mean to Use? — Decision Framework

Situation / Question Use This Mean Why
Estimating average return for a single future period Arithmetic Arithmetic mean = average of one-period returns
Estimating average returns over multiple future periods / historical performance Geometric Captures compounding; shows compound rate of growth
Averaging ratios or rates (e.g., P/E ratios, prices in cost averaging) Harmonic Appropriate for "amount per unit" ratios; reduces outlier impact
Extreme outliers present — want to EXCLUDE them Trimmed Removes extreme observations before averaging
Extreme outliers present — want to INCLUDE but dampen them Winsorized Replaces extremes with nearest non-extreme values
No outliers; symmetric distribution; compounding irrelevant Arithmetic Simplest; efficient; known statistical properties
The Inequality — Always True (Unless All Returns Are Equal)
Harmonic Mean ≤ Geometric Mean ≤ Arithmetic Mean
All three means are equal only when all observations in the dataset are identical.
The gap between arithmetic and geometric means increases with greater variability (dispersion) in the returns.
The arithmetic mean is always ≥ geometric mean — it is biased upward as a measure of multi-period performance.
06

Money-Weighted Return (MWR)

Money-Weighted Return (MWR) = Internal Rate of Return (IRR)
The discount rate that makes the sum of present values of all cash flows equal to zero. Reflects the actual return earned on the actual money invested, accounting for the size and timing of cash flows. Equivalent to a bond's yield to maturity calculation.
Money-Weighted Return — IRR Formula
Σ [CFₜ / (1 + IRR)ᵗ] = 0 for t = 0 to T

CF convention (from investor's perspective):
Outflows (money leaving investor) = NEGATIVE: −purchases, −additional investments
Inflows (money coming to investor) = POSITIVE: +dividends, +proceeds from sales, +ending value
Solve using financial calculator or Excel =IRR(values) function.
The MWR gives the annualized return on the actual dollars invested for the entire period.

Step-by-Step MWR Process

  1. Identify all cash flows by time period — investments (outflows, negative) and receipts (inflows, positive)
  2. Net cash flows at each time point (e.g., at t=1: −new purchase + dividend received)
  3. Include ending portfolio value as a positive cash flow at the final period
  4. Solve for IRR using calculator or Excel =IRR()

Worked Example — Dividend-Paying Stock

Buy 1 share at t=0 for USD200. Buy 1 more at t=1 for USD225. Sell both at t=2 for USD235/share. Dividend = USD5/share/year. t=1 dividend not reinvested.

  • CF₀ = −200 (buy 1 share)
  • CF₁ = −225 + 5 = −220 (buy 2nd share, receive dividend on 1st share)
  • CF₂ = +470 + 10 = +480 (sell 2 shares × $235, receive dividends on 2 shares)

Solve: −200/(1+IRR)⁰ + (−220)/(1+IRR)¹ + 480/(1+IRR)² = 0 → MWR = 9.39%

Individual HPRs: Year 1 = (5+225−200)/200 = 15%. Year 2 = (10+470−450)/450 = 6.67%. Mean HPR = 10.84%.

MWR (9.39%) < Mean HPR (10.84%) because more money was invested in Year 2 when returns were lower (6.67%) → MWR down-weights good performance, up-weights poor performance when more is invested at that time.

⚠️ Limitation of MWR — Not Comparable Across Investors

Two investors in the same fund can have different MWRs simply because they invested different amounts at different times. MWR is not suitable for comparing portfolio manager skill — it reflects the investor's cash flow decisions, not the manager's investment decisions.

07

Time-Weighted Return (TWR)

Time-Weighted Return (TWR)
Measures the compound rate of growth of USD1 (or any single unit of currency) invested in the portfolio over a stated measurement period. NOT sensitive to the timing and amount of cash flows. The preferred performance measure for evaluating portfolio managers because cash withdrawals/additions are generally outside the manager's control.

3-Step Process for Exact TWR

  1. Price the portfolio immediately before any significant cash flow (addition or withdrawal). Break the overall evaluation period into sub-periods based on cash flow dates.
  2. Calculate the HPR for each sub-period using: HPR = (Ending value + CF out − CF in) / Beginning value
  3. Link (compound) the sub-period HPRs:
    • For a single year: TWR = [(1+HPR₁)(1+HPR₂)...(1+HPRₙ)] − 1
    • For multiple years: take the geometric mean of the annual TWRs
TWR Formulas
TWR (single year, n sub-periods):
TWR = [(1+HPR₁)(1+HPR₂)...(1+HPRₙ)] − 1

TWR (annualized over N years):
R_TW = [(1+R₁)(1+R₂)...(1+Rₙ)]^(1/N) − 1

Using our dividend stock example (HPR₁ = 15%, HPR₂ = 6.67%):
TWR = √[(1.15)(1.0667)] − 1 = √(1.2267) − 1 = 10.76%
TWR (10.76%) > MWR (9.39%) in this example because more money was invested in the lower-return year.
Daily portfolio valuation is standard practice — the more frequent the valuation, the more accurate the TWR approximation.
08

MWR vs. TWR — The Critical Comparison

Feature Money-Weighted Return (MWR) Time-Weighted Return (TWR)
Also known as Internal Rate of Return (IRR) Geometric mean of sub-period HPRs
Sensitive to cash flow timing? YES — heavily influenced by size & timing NO — neutralizes cash flow effects
What it measures Actual return earned on actual money invested Compound growth rate of USD1 invested throughout
Best used for What the investor actually earned Evaluating portfolio manager skill
Can two investors in same fund differ? YES — different cash flow timing → different MWR NO — same TWR regardless of cash flows
Effect of investing MORE at a bad time MWR is depressed (hurt) TWR is unaffected
Effect of investing MORE at a good time MWR is elevated (helped) TWR is unaffected
Calculation method Solve for IRR (trial and error / Excel) Geometric mean of linked HPRs
💡 Exam Shortcut — MWR vs TWR Direction

If MORE money is invested before a period of POOR performance → MWR < TWR.
If MORE money is invested before a period of GOOD performance → MWR > TWR.
If cash flows are small or returns are similar → MWR ≈ TWR.

Annualizing Returns

General Annualization Formula
R_annual = (1 + R_period)^c − 1

where c = number of periods in one year

Period conventions:
Daily → c = 365 (calendar) or ~250 (trading days)
Weekly → c = 52
Monthly → c = 12
Quarterly → c = 4
18-month → c = 2/3 (since 1 year = 2/3 of an 18-month period)
PV formula with non-annual compounding: PV = FV_N × (1 + Rs/m)^(−mN)
where m = compounding periods/year, Rs = quoted annual rate, N = years

Worked Examples

  • Weekly return of 0.2%: R_annual = (1.002)⁵² − 1 = 10.95%
  • 15-day return of 0.4%: R_annual = (1.004)^(365/15) − 1 = 10.20%
  • 18-month return of 20%: R_annual = (1.20)^(2/3) − 1 = 12.92%
  • Annual → weekly: R_weekly = (1 + R_annual)^(1/52) − 1
  • Daily → weekly (5 trading days): R_weekly = (1 + R_daily)⁵ − 1
⚠️ Limitation of Annualizing Short-Period Returns

Annualizing assumes returns can be repeated precisely every period. This is unrealistic for short-period returns. A 5% weekly return → annualized = 1,164% — clearly unrealistic. Use annualized short-term returns for comparison purposes only, with this limitation in mind.

10

Continuously Compounded Returns

Continuously Compounded Return (r)
The natural logarithm of 1 plus the holding period return, or equivalently, the natural log of the ending price divided by the beginning price (the "price relative"). As the number of compounding periods approaches infinity, this is the limiting value. Always LESS than the associated HPR.
Continuously Compounded Return Formulas
One-period: r(t,t+1) = ln(P(t+1) / Pₜ) = ln(1 + R(t,t+1))

Multi-period: r(0,T) = ln(Pₜ / P₀)

Recover ending price: Pₜ = P₀ × exp(r(0,T))

KEY PROPERTY — Additive over time:
r(0,T) = r(T−1,T) + r(T−2,T−1) + ... + r(0,1)
(Sum of continuously compounded returns = total continuously compounded return)
Continuously compounded return is always SMALLER than HPR: ln(1.04) = 0.0392 < 4%
Using HPRs requires MULTIPLICATION of (1+R) terms. Using cc returns requires only ADDITION — a desirable mathematical property.
Example: P₀ = USD30, P₁ = USD34.50 → cc return = ln(34.50/30) = ln(1.15) = 13.98%
💡 Why Continuous Compounding Matters

The additive property of continuously compounded returns is mathematically convenient — it means multi-period cc returns are simply the sum of single-period cc returns. This makes them useful in options pricing (Black-Scholes), risk models, and statistical analysis throughout the curriculum.

Other Major Return Measures

Gross Return

Return on assets managed less trading expenses and commissions only. Does NOT deduct management fees, custody fees, or taxes (these vary by investor). Used to evaluate and compare investment manager skill.

Net Return

Gross return minus ALL managerial and administrative expenses. What the investor actually receives. Small funds are disadvantaged vs. large funds (fixed admin costs spread over smaller asset base → many small funds waive expenses to compete).

Pre-Tax Nominal Return

The DEFAULT — all returns are pre-tax nominal unless otherwise stated. No adjustment for taxes or inflation.

After-Tax Nominal Return

Total return minus taxes on dividends, interest, and realized capital gains. Taxable investors evaluate managers on after-tax returns. Tax treatment varies: long-term capital gains often receive preferential rates; interest = ordinary income in most jurisdictions; dividends vary by country.

Nominal Return

Return NOT adjusted for inflation. Default return unless stated otherwise.

Real Return

Return adjusted for inflation. Useful for comparing returns across time periods (inflation varies) and across countries (different local inflation rates). Formula: (1 + nominal) = (1 + real)(1 + inflation).

Nominal vs. Real Return — Key Formulas
Exact: (1 + nominal risk-free) = (1 + real risk-free)(1 + inflation premium)
Approx: Nominal ≈ Real + Inflation

For risky assets:
(1 + nominal return) = (1 + real return)(1 + risk premium)

Note: Real return = nominal return adjusted for inflation only
After-tax real return = what investor truly receives after taxes AND inflation
(Not commonly calculated by managers — depends on each investor's marginal tax rate, holding period, account type)

Leveraged Return

Leveraged Return (RL)
Return on the investor's own equity when borrowed funds are used to amplify the position. If portfolio return > borrowing cost → leverage increases return. If portfolio return < borrowing cost → leverage decreases return. Both gains AND losses are amplified.
Leveraged Return Formula
RL = RP + (VB / VE) × (RP − rD)

where:
RP = total portfolio return on leveraged portfolio
VB = borrowed funds (debt)
VE = investor's own equity
rD = borrowing cost (interest rate on debt)

Example: EUR10M portfolio, 8% return, financed 30% debt at 5%
→ VB = EUR3M, VE = EUR7M
→ RL = 8% + (3/7)(8% − 5%) = 8% + 0.43 × 3% = 8% + 1.29% = 9.29%
Two ways to create leverage: (1) Futures contracts — margin as little as 10% of notional, so return is up to 10× amplified.
(2) Borrowing to invest — common in stocks, bonds, real estate.
If RP > rD → leverage is positive (enhances return). If RP < rD → leverage is negative (destroys return).
⚠️ Leverage Amplifies BOTH Gains AND Losses

Leverage does not selectively amplify gains. If the portfolio loses money and the borrowing cost must still be paid, the loss to the investor's equity can be far greater than the portfolio loss itself. Futures contracts can amplify by a factor of 10× — a 5% adverse move becomes a 50% loss on the margin posted.

  • ✅ Nominal risk-free rate (exact): (1 + nominal) = (1 + real)(1 + inflation). Approx: nominal ≈ real + inflation (underestimates slightly). Government short-term debt = nominal risk-free rate proxy for that country.
  • ✅ Liquidity premium: T-bills have NONE (too liquid). Small issuer bonds have liquidity premiums. Maturity premium: compensates for price sensitivity of longer-maturity bonds to interest rate changes.
  • ✅ Holding period return: R = (P₁ − P₀ + I₁) / P₀. Two components: capital gain (P₁−P₀)/P₀ + income yield I₁/P₀. Multi-period: R = [(1+R₁)(1+R₂)...(1+Rₜ)] − 1. Never multiply returns directly — always use (1+R) format.

  • ✅ Geometric mean: Compound rate of growth; best for multi-period performance; what you actually earn. Formula: R̄G = [(1+R₁)(1+R₂)...(1+Rₜ)]^(1/T) − 1. Gap vs. arithmetic mean increases with return variability.
  • ✅ Harmonic mean: n / Σ(1/Xᵢ). For rates and ratios (P/E, cost averaging). Weights each observation inversely to its magnitude → reduces outlier impact. Only for non-negative values.

  • ✅ Non-annual compounding PV: PV = FV × (1 + Rs/m)^(−mN). Rs = quoted annual rate, m = periods/year, N = years. Periodic rate = Rs/m; total periods = mN.
  • ✅ Continuously compounded return: r(t,t+1) = ln(P(t+1)/Pₜ) = ln(1 + HPR). Always LESS than HPR (e.g., HPR=4% → cc return = ln(1.04) = 3.92%). Key property: cc returns are ADDITIVE over time (vs. HPRs which are multiplicative). P_T = P₀ × e^(r₀,T).
  • ✅ Gross return: Less trading costs only; for comparing manager skill. Net return: Less ALL expenses; what investor actually earns. Small funds disadvantaged vs. large funds (fixed admin costs ÷ fewer assets).
  • ✅ Default return type: All returns are pre-tax nominal UNLESS stated otherwise.
  • ✅ Real return: (1 + nominal) = (1 + real)(1 + inflation). Useful to compare across time periods (variable inflation) and across countries (different local inflation). After-tax real return = what investor truly earns after inflation AND taxes.
  • ✅ Leveraged return: RL = RP + (VB/VE)(RP − rD). If RP > rD → leverage amplifies gains. If RP < rD → leverage amplifies losses. Futures: margin as low as 10% → 10× amplification of gains AND losses.
