Module Overview & Key Concepts

This module answers one fundamental question: Is there a statistically significant relationship between two variables? Two different scenarios require two different approaches:

  • Continuous / numerical variables → test the correlation coefficient using a t-test (parametric: Pearson) or rank-based t-test (nonparametric: Spearman).
  • Categorical / discrete variables → test independence using a chi-square test on a contingency table (always nonparametric).
Parametric Test of Correlation
Uses the Pearson (bivariate) correlation coefficient r. Requires that both variables are normally distributed. Test statistic is t-distributed with n−2 degrees of freedom. Tests H₀: ρ = 0 (no linear relationship).
Nonparametric Test of Correlation
Uses the Spearman rank correlation coefficient rₛ. Calculated on ranks of observations, not raw values. Used when normality assumption is violated, data has outliers, or data is ordinal. Same t-test formula as Pearson for large samples (n > 30).
Test of Independence (Contingency Table)
Uses a chi-square test statistic on a contingency (two-way) table. For categorical/discrete data where correlation is not applicable. Tests whether two categorical classifications are independent of each other. Always a right-tail test. Degrees of freedom = (r−1)(c−1).
💡 Big Picture: When to Use What

Pearson (parametric): Two continuous variables, both approximately normal. Spearman (nonparametric): Two variables with non-normal distribution, outliers, or ordinal/ranked data. Chi-square contingency test: Two categorical/discrete variables — e.g., fund size (small/medium/large) × investment type (value/growth/blend). You CANNOT use correlation when data is categorical.

Pearson Correlation — The Parametric Approach

Sample Pearson Correlation Coefficient
r_XY = s_XY / (s_X × s_Y)

where:
s_XY = sample covariance between X and Y
s_X = sample standard deviation of X
s_Y = sample standard deviation of Y
Also called pairwise correlation, bivariate correlation, or simply "the correlation." Often written as just r.
Range: −1 ≤ r ≤ +1. A value of +1 = perfect positive linear relationship; −1 = perfect negative; 0 = no linear relationship.
The covariance drives the sign of the correlation — positive covariance → positive r; negative covariance → negative r.
Each observation is compared to its respective variable mean — how much each observation differs from its mean matters.
This formula applies when both X and Y are approximately normally distributed.

Hypotheses for Correlation Tests

Let ρ = population correlation coefficient. The three possible hypothesis set-ups:

  • Two-sided:      H₀: ρ = 0   vs.  Hₐ: ρ ≠ 0    ← most common; tests if ANY relationship exists
  • Right-sided:    H₀: ρ ≤ 0   vs.  Hₐ: ρ > 0    ← tests positive relationship specifically
  • Left-sided:     H₀: ρ ≥ 0   vs.  Hₐ: ρ < 0    ← tests negative relationship specifically

We use the sample statistic r to test hypotheses about the population parameter ρ.

03

t-test for the Correlation Coefficient

t-Statistic for Testing H₀: ρ = 0 (Parametric — Pearson)
t = r × √(n − 2) / √(1 − r²)

Degrees of freedom: df = n − 2
r = sample correlation coefficient (Pearson or Spearman — same formula for both with large n).
n = number of paired observations (pairs of X, Y values).
The −2 in degrees of freedom: we estimate two parameters (mean of X, mean of Y) from the data, costing 2 degrees of freedom.
Under H₀: ρ = 0, this statistic follows a t-distribution with n−2 df, assuming both variables are normally distributed.
Reject H₀ at significance level α: if |t| > t_critical (two-tailed) or t > t_critical / t < −t_critical (one-tailed).
📐 Worked Example: Testing Correlation — Two Funds

Setup: Analyst tests whether returns of Fund A and Fund B are correlated over n = 35 years. Uses Spearman rank correlation (non-normal data). Finds rₛ = 0.6916. Test at α = 5%, two-tailed.

H₀: rₛ = 0  vs.  Hₐ: rₛ ≠ 0.    df = 35 − 2 = 33.    Critical values: ±2.0345.

t = 0.6916 × √(35 − 2) / √(1 − 0.6916²)
t = 0.6916 × √33 / √(1 − 0.4783)
t = 0.6916 × 5.7446 / √0.5217
t = 3.9727 / 0.7223
t = 5.5005
|5.5005| > 2.0345 → REJECT H₀

Conclusion: Sufficient evidence that the Spearman rank correlation between Fund A and Fund B returns is significantly different from zero — the funds' returns are correlated.

⚠️ Exam Trap: Always df = n − 2 for Correlation Tests

For testing correlation (both Pearson and Spearman with large n), degrees of freedom = n − 2, NOT n − 1. This is a common mistake. The −2 comes from estimating two means from the data. Contrast with the single-mean t-test where df = n−1.

04

Effect of Sample Size on Correlation Tests

The magnitude of r needed to reject H₀: ρ = 0 decreases as sample size n increases. Two reasons:

  • Reason 1: As n increases → df = n−2 increases → the t-distribution becomes narrower → the absolute value of the critical t-value decreases → easier to exceed it.
  • Reason 2: As n increases → the numerator r√(n−2) increases in absolute value (for the same r) → the calculated t-statistic is larger in magnitude.
Numerical Example — Same r = 0.35, Different Sample Sizes
n = 12, r = 0.35: t = 0.35×√10 / √(1−0.35²) = 1.182 < critical ±2.228 → NOT significant
n = 32, r = 0.35: t = 0.35×√30 / √(1−0.35²) = 2.046 > critical ±2.042 → JUST significant
Same sample correlation (r = 0.35), but significance changes purely because of sample size. This illustrates why statistical significance must always be interpreted alongside practical/economic significance — with very large datasets, even tiny correlations become statistically significant but may be meaningless in practice.
⚠️ Exam Trap: Big Data Problem — Spurious Significance

As datasets grow very large, the null hypothesis of zero correlation is almost always rejected, even for economically trivial relationships. A correlation of r = 0.02 could be statistically significant with n = 10,000 observations but is likely meaningless in practice. The exam may test whether you understand that statistical significance ≠ economic significance, and that with large samples, additional analytical tools must be applied beyond simple hypothesis testing of correlation.

💡 Higher r → Reaches Significance Faster

As you increase sample size, higher correlations achieve statistical significance at smaller n than lower correlations do. A correlation of r = 0.8 requires far fewer observations to be declared significant than r = 0.2. The rate at which significance is achieved as n grows depends on the magnitude of r — this is about the power of the test increasing with both n and the true effect size.

05

Spearman Rank Correlation — The Nonparametric Approach

The Spearman rank correlation coefficient (rₛ) is the nonparametric alternative to Pearson correlation. It is essentially the Pearson correlation calculated on the ranks of the observations rather than their actual values. It measures the strength of the monotonic (not just linear) relationship between two variables.

Spearman Rank Correlation Formula
rₛ = 1 − [ 6 × Σᵢ dᵢ² ] / [ n × (n² − 1) ]

where dᵢ = difference in ranks of the i-th pair (rank of Xᵢ minus rank of Yᵢ)
dᵢ = rank(Xᵢ) − rank(Yᵢ) for each paired observation i.
n = number of paired observations.
Range: −1 ≤ rₛ ≤ +1, same interpretation as Pearson r.
If all pairs have identical ranks on both variables → all dᵢ = 0 → Σdᵢ² = 0 → rₛ = 1 (perfect positive monotonic relationship).
If ranks are perfectly reversed → rₛ = −1.

Step-by-Step: How to Calculate rₛ

1
Rank variable X from largest to smallest: assign rank 1 to the largest, rank 2 to the second largest, etc. Handle ties: assign the average of the ranks they jointly occupy. E.g., if the 3rd and 4th largest values are tied → both get rank 3.5 (average of 3 and 4). Repeat for variable Y independently.
2
Calculate dᵢ = rank(Xᵢ) − rank(Yᵢ) for each paired observation. Then calculate dᵢ² (squared difference in ranks) for each pair.
3
Sum the squared differences: compute Σdᵢ². Substitute into the formula: rₛ = 1 − [6 × Σdᵢ²] / [n(n²−1)].
4
Test significance: For large samples (n > 30), use the same t-statistic as Pearson: t = rₛ√(n−2) / √(1−rₛ²) with df = n−2. For small samples (n ≤ 30), use specialized Spearman critical value tables.

When to Use Spearman Instead of Pearson

Use Spearman When…
  • Population departs meaningfully from normality (non-normal distribution).
  • Data contains extreme values or outliers that would distort Pearson r.
  • Data is measured on an ordinal/ranked scale (e.g., analyst ratings, credit quality rankings).
  • You want to detect a monotonic (not necessarily linear) relationship.
  • Small sample from unknown distribution.
Use Pearson When…
  • Both variables are approximately normally distributed.
  • Data is on interval or ratio scale (continuous).
  • You want to detect a specifically linear relationship.
  • No significant outliers in the data.
  • Parametric assumptions are met — Pearson is more powerful when valid.
🧠 Memory Trick: Spearman vs. Pearson

Pearson = "Pure" data — works on raw values, requires pure normality assumption. Spearman = "Safe" choice — works on ranks, safer when normality is questionable. Both use the same t-test formula (t = r√(n−2)/√(1−r²)) for large n — the only difference is whether r is the raw correlation or the rank correlation. Spearman with n ≤ 30 → need special tables.

Contingency Tables — Setup & Terminology

Contingency Table (Two-Way Table)
A table that displays frequencies (counts) of observations classified by two categorical variables simultaneously. Rows represent categories of one variable; columns represent categories of another. Used when correlation methods are not applicable (categorical/discrete data).

Example from the module: 1,594 ETFs classified by two dimensions — Size (small, medium, large) and Investment Type (value, growth, blend). We cannot compute a correlation between "size category" and "investment type" because these are categorical labels, not numeric values. Instead, we use a chi-square test of independence.

The ETF Contingency Table (from the Module)

Investment Type Small Cap Medium Cap Large Cap Row Total
Value 50 110 343 503
Growth 42 122 202 366
Blend 56 149 520 725
Column Total 148 381 1,065 1,594
Expected Frequency Under Independence
Eᵢⱼ = (Row i Total × Column j Total) / Overall Total

Example: E(Value, Small) = (503 × 148) / 1,594 = 74,444 / 1,594 = 46.70
Eᵢⱼ = expected count in cell (i, j) if the two variables were completely independent.
Logic: if size and investment type are independent, the proportion of ETFs in each cell should simply equal the product of the marginal proportions (row proportion × column proportion × total).
The test statistic compares these expected frequencies to the actual (observed) frequencies Oᵢⱼ.
07

Chi-Square Test of Independence

Chi-Square Test Statistic for Independence (Equation 4)
χ² = Σᵢ Σⱼ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ

Degrees of freedom: df = (r − 1) × (c − 1)

r = number of rows (categories of first variable)
c = number of columns (categories of second variable)
Oᵢⱼ = observed frequency in cell (i, j) — what we actually count in the data.
Eᵢⱼ = expected frequency in cell (i, j) = (row i total × column j total) / overall total.
m = total number of cells = r × c (summation runs over all m cells).
If Oᵢⱼ = Eᵢⱼ for all cells → χ² = 0 (perfect independence). Larger deviations → larger χ² → stronger evidence of dependence.
Differences are SQUARED → χ² is always ≥ 0 → always a right-tail test only (one rejection region on the right).

Hypotheses for the Chi-Square Test of Independence:

  • H₀: The two categorical variables ARE independent (no relationship).
  • Hₐ: The two categorical variables are NOT independent (a relationship exists).

Decision rule: Reject H₀ if χ² > χ²_critical (right tail, df = (r−1)(c−1), significance level α).

Full Six-Step Worked Example: ETF Size vs. Investment Type

1
State the hypotheses: H₀: ETF size and investment type are independent (not related). Hₐ: ETF size and investment type are dependent (related).
2
Test statistic: χ² = Σ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ — chi-square distributed.
3
Level of significance: α = 5%.
4
Decision rule: df = (3−1) × (3−1) = 2 × 2 = 4. Critical value at 5% with 4 df = 9.4877. Reject H₀ if χ² > 9.4877 (right-tail only).
5
Calculate: First compute all 9 expected frequencies Eᵢⱼ = (row total × column total) / 1,594. Then sum (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ across all 9 cells. Calculated χ² = 32.08025.
6
Decision: 32.08 > 9.4877 → Reject H₀. Sufficient evidence that ETF size and investment type are related (not independent) at the 5% level.
📐 How to Compute Expected Frequencies — Key Examples

Using the ETF table: Overall total = 1,594. Row totals: Value=503, Growth=366, Blend=725. Column totals: Small=148, Medium=381, Large=1,065.

E(Value, Small) = (503 × 148) / 1,594 = 74,444 / 1,594 = 46.70
E(Value, Medium) = (503 × 381) / 1,594 = 191,643 / 1,594 = 120.23
E(Value, Large) = (503 × 1065) / 1,594 = 535,695 / 1,594 = 336.07
E(Growth, Small) = (366 × 148) / 1,594 = 54,168 / 1,594 = 33.98
E(Blend, Large) = (725 × 1,065)/ 1,594 = 772,125 / 1,594 = 484.40
Then: χ² = Σ (O−E)²/E across all 9 cells = 32.08025

Standardized Residual (Pearson Residual)

Standardized Residual for Cell (i,j)
Standardized residual = (Oᵢⱼ − Eᵢⱼ) / √Eᵢⱼ
Also called the Pearson residual. Measures how much each individual cell contributes to the overall chi-square statistic, standardized for comparison across cells.
Positive residual: Observed > Expected — more observations in this cell than independence would predict (dark shading in mosaic plots).
Negative residual: Observed < Expected — fewer observations than expected under independence (light shading).
Note: (Standardized residual)² = the contribution of cell (i,j) to the χ² statistic. This helps identify WHICH cells are driving the departure from independence.
💡 Chi-Square Independence Test — Always Right-Tail Only

Unlike chi-square tests for a single variance (which can be left-tail, right-tail, or two-tailed), the chi-square test of independence is always right-tailed only. This is because (O−E)² is always positive — if observed and expected are equal, χ² = 0; any deviation (in either direction) increases χ². We can only reject H₀ when χ² is sufficiently large.

Pearson r — Pros & Cons
  • ✅ More statistically powerful when normality holds.
  • ✅ Standard, widely used, easy to interpret.
  • ❌ Sensitive to outliers — one extreme value can flip the sign of r.
  • ❌ Requires normality — invalid for heavily skewed data.
  • ❌ Only detects LINEAR relationships; misses curved/monotonic patterns.
Spearman rₛ — Pros & Cons
  • ✅ Robust to outliers and non-normality.
  • ✅ Works for ordinal/ranked data.
  • ✅ Detects any monotonic relationship (not just linear).
  • ❌ Less powerful than Pearson when normality holds.
  • ❌ Small samples (n ≤ 30) require specialized tables.
  • ❌ Loses information by converting to ranks.
★

Exam Checklist — Must-Know Points

  • ✅ ρ = population correlation; r = sample correlation (Pearson) or rₛ (Spearman). We test hypotheses about ρ using r.
  • ✅ Most common hypothesis: H₀: ρ = 0 vs. Hₐ: ρ ≠ 0 (two-sided). Tests whether any linear relationship exists at all. Can also be one-sided if testing direction.
  • ✅ t-test for correlation: t = r√(n−2) / √(1−r²), df = n−2. Used for BOTH Pearson (parametric) and Spearman (nonparametric, large n only).
  • ✅ df = n−2 for correlation tests — not n−1. This is the most common degree-of-freedom mistake on this topic.
  • ✅ Pearson correlation formula: r = s_XY / (s_X × s_Y). The covariance drives the sign. Requires both variables to be approximately normally distributed.
  • ✅ Larger n → easier to reject H₀: ρ = 0 for the same sample r. Two reasons: (1) df increases → critical t decreases; (2) numerator r√(n−2) increases. With very large datasets, even tiny correlations become statistically significant — always also assess economic significance.
  • ✅ Spearman rank correlation: rₛ = 1 − [6Σdᵢ²] / [n(n²−1)] where dᵢ = rank(Xᵢ) − rank(Yᵢ).
  • ✅ Spearman is used when: non-normal population, outliers present, data is ordinal/ranked, or when checking monotonic (not just linear) relationships.
  • ✅ Ranking ties: assign the average of the ranks they jointly occupy. E.g., if 3rd and 4th are tied → both get rank 3.5.
  • ✅ Small samples (n ≤ 30): Spearman requires specialized critical value tables — NOT the standard t-table. Large samples (n > 30): use the standard t-test.
  • ✅ Contingency table: Used when data is categorical/discrete. You CANNOT use correlation for categorical variables.
  • ✅ Chi-square independence test: χ² = Σ(Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ. Always right-tail only (squared differences → always positive).
  • ✅ Expected frequency formula: Eᵢⱼ = (Row i total × Column j total) / Overall total. This is the count you'd expect if the two variables were completely independent.
  • ✅ Chi-square independence df: (r−1)(c−1) where r = number of row categories, c = number of column categories. ETF example: (3−1)(3−1) = 4.
  • ✅ Decision rule: Reject H₀ of independence if calculated χ² > right-tail critical value.
  • ✅ Standardized residual = (Oᵢⱼ − Eᵢⱼ) / √Eᵢⱼ. Positive = more than expected; negative = less than expected. Squaring gives each cell's contribution to χ².
  • ✅ When to use which test: Continuous + normal → Pearson t-test. Continuous + non-normal or ordinal → Spearman t-test. Categorical → chi-square contingency test.
